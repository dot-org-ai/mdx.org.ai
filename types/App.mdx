# App

Root application wrapper that provides global context, theme, and error boundaries.

## Structure

```
┌─────────────────────────────────────────┐
│ root                                    │
│ ┌─────────────────────────────────────┐ │
│ │ providers                           │ │
│ │ ┌─────────────────────────────────┐ │ │
│ │ │ error-boundary                  │ │ │
│ │ │                                 │ │ │
│ │ │   {children}                    │ │ │
│ │ │                                 │ │ │
│ │ └─────────────────────────────────┘ │ │
│ └─────────────────────────────────────┘ │
└─────────────────────────────────────────┘
```

## Usage

```mdx
<App appName="My Application" theme="system" locale="en">
  <Site>
    <Page>
      <Hero title="Welcome" />
    </Page>
  </Site>
</App>
```
