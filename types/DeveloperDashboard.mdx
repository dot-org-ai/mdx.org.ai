## Features

- **API Key Management** - Create, rotate, and revoke API keys
- **Usage Metrics** - Real-time API usage with charts and breakdowns
- **Request Logs** - Searchable log viewer with filters
- **Webhooks** - Configure and test webhook endpoints
- **Organization Switching** - Multi-org support
