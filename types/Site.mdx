# Site

A complete website wrapper providing consistent header, navigation, and footer across all pages.

## Structure

```
┌─────────────────────────────────────────┐
│ header                                  │
│ ┌──────┐ ┌────────────────────────────┐ │
│ │ logo │ │ nav                        │ │
│ └──────┘ └────────────────────────────┘ │
├─────────────────────────────────────────┤
│                                         │
│ main                                    │
│                                         │
│   {children}                            │
│                                         │
├─────────────────────────────────────────┤
│ footer                                  │
│ ┌─────────────┐ ┌─────────┐ ┌─────────┐ │
│ │ footer-nav  │ │ social  │ │copyright│ │
│ └─────────────┘ └─────────┘ └─────────┘ │
└─────────────────────────────────────────┘
```

## Usage

```mdx
<Site
  siteName="Acme Corp"
  logo="/logo.svg"
  nav={[
    { label: 'Home', href: '/' },
    { label: 'Products', href: '/products' },
    { label: 'About', href: '/about' },
  ]}
  copyright="© 2024 Acme Corp"
>
  <Page title="Welcome">
    <Hero title="Build something great" />
  </Page>
</Site>
```

```html

  <header part="header">
    <a part="logo" href="/"><img src="/logo.svg" alt="Acme" /></a>

    </nav>
  </header>
  <main part="main">{children}</main>
  <footer part="footer">
    <nav part="footer-nav">...</nav>
    <div part="social">...</div>
    <p part="copyright">© 2024</p>
  </footer>
</div>
```
