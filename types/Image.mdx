# Image

Optimized responsive image component with lazy loading, placeholders, and optional zoom.

## Structure

```
┌─────────────────────────────────────────────────────┐
│ root                                                │
│ ┌─────────────────────────────────────────────────┐ │
│ │ container                                       │ │
│ │ ┌─────────────────────────────────────────────┐ │ │
│ │ │ picture                                     │ │ │
│ │ │ ┌─────────────────────────────────────────┐ │ │ │
│ │ │ │ img                                     │ │ │ │
│ │ │ │                                         │ │ │ │
│ │ │ │          [image content]                │ │ │ │
│ │ │ │                                         │ │ │ │
│ │ │ └─────────────────────────────────────────┘ │ │ │
│ │ │ placeholder (while loading)                 │ │ │
│ │ └─────────────────────────────────────────────┘ │ │
│ └─────────────────────────────────────────────────┘ │
│ ┌─────────────────────────────────────────────────┐ │
│ │ caption                          credit: Photo │ │
│ └─────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────┘
```

## Usage

### Basic Image

```mdx
<Image
  src="/hero.jpg"
  alt="Hero image"
  width={1200}
  height={600}
/>
```

### Responsive Image

```mdx
<Image
  src="/photo.jpg"
  alt="Landscape photo"
  sizes="(max-width: 768px) 100vw, 50vw"
  srcSet="/photo-400.jpg 400w, /photo-800.jpg 800w, /photo-1200.jpg 1200w"
/>
```

### With Blur Placeholder

```mdx
<Image
  src="/product.jpg"
  alt="Product shot"
  placeholder="blur"
  blurDataURL="data:image/jpeg;base64,/9j/4AAQSkZJRg..."
  width={800}
  height={600}
/>
```

### With Caption and Credit

```mdx
<Image
  src="/landscape.jpg"
  alt="Mountain landscape"
  caption="View from Mount Everest base camp"
  credit="Photo by John Doe"
/>
```

### Multiple Sources (Art Direction)

```mdx
<Image
  src={[
    { src: '/hero-mobile.jpg', media: '(max-width: 767px)' },
    { src: '/hero-tablet.jpg', media: '(max-width: 1023px)' },
    { src: '/hero-desktop.jpg', media: '(min-width: 1024px)' },
  ]}
  alt="Responsive hero image"
/>
```

### With Zoom on Click

```mdx
<Image
  src="/detailed-chart.png"
  alt="Analytics chart"
  zoom
/>
```
