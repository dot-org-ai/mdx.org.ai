# Screenshot

Renders any component to a static image (SVG, PNG, etc.) for use in documentation, social cards, or marketing materials.

## Structure

```
┌─────────────────────────────────────────────────────┐
│ root                                                │
│ ┌─────────────────────────────────────────────────┐ │
│ │ frame (optional device frame)                   │ │
│ │ ┌─────────────────────────────────────────────┐ │ │
│ │ │ frame-toolbar                               │ │ │
│ │ │ ● ● ●            title           ↔         │ │ │
│ │ └─────────────────────────────────────────────┘ │ │
│ │ ┌─────────────────────────────────────────────┐ │ │
│ │ │ viewport                                    │ │ │
│ │ │ ┌─────────────────────────────────────────┐ │ │ │
│ │ │ │ content                                 │ │ │ │
│ │ │ │                                         │ │ │ │
│ │ │ │   <Dashboard {...props} />              │ │ │ │
│ │ │ │                                         │ │ │ │
│ │ │ │                                         │ │ │ │
│ │ │ └─────────────────────────────────────────┘ │ │ │
│ │ └─────────────────────────────────────────────┘ │ │
│ └─────────────────────────────────────────────────┘ │
│ ┌─────────────────────────────────────────────────┐ │
│ │ caption                                         │ │
│ └─────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────┘
```

## Usage

### Basic Screenshot

```mdx
<Screenshot
  component={Dashboard}
  props={{
    title: 'Analytics Dashboard',
    stats: [
      { label: 'Users', value: '12,345' },
      { label: 'Revenue', value: '$45,678' },
    ],
  }}
  width={1200}
  height={800}
  format="png"
/>
```

### With Browser Frame

```mdx
<Screenshot
  component={Site}
  props={{
    siteName: 'Acme Corp',
    nav: [{ label: 'Home', href: '/' }],
    children: <LandingPage {...landingProps} />,
  }}
  deviceFrame="browser"
  shadow="lg"
  borderRadius={12}
  caption="Homepage design"
/>
```

### Mobile Screenshot

```mdx
<Screenshot
  component={App}
  props={mobileAppProps}
  width={390}
  height={844}
  deviceFrame="mobile"
  theme="dark"
/>
```

### Social Card / OG Image

```mdx
<Screenshot
  component={SocialCard}
  props={{
    title: 'How to Build Better APIs',
    author: { name: 'Alice', avatar: '/alice.jpg' },
    date: '2024-03-15',
  }}
  width={1200}
  height={630}
  format="png"
  alt="Blog post social card"
/>
```

## Output Formats

| Format | Use Case |
|--------|----------|
| `svg` | Scalable, editable, web |
| `png` | Screenshots, transparency |
| `jpg` | Photos, smaller files |
| `webp` | Modern web, best compression |
