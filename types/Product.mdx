# Product

Product card for displaying items in e-commerce, marketplaces, and product catalogs.

## Structure

### Card Layout (Default)

```
┌─────────────────────────────────────┐
│ root                                │
│ ┌─────────────────────────────────┐ │
│ │ image-container                 │ │
│ │ ┌─────────────────────────────┐ │ │
│ │ │ image                       │ │ │
│ │ │                             │ │ │
│ │ │                             │ │ │
│ │ └─────────────────────────────┘ │ │
│ │ ┌──────┐              ┌──────┐  │ │
│ │ │ badge│              │  ♡   │  │ │
│ │ │ NEW  │              │wish  │  │ │
│ │ └──────┘              └──────┘  │ │
│ └─────────────────────────────────┘ │
│ ┌─────────────────────────────────┐ │
│ │ content                         │ │
│ │ brand / category                │ │

│ │ │ price                       │ │ │
```

## Usage

```mdx
<Product
  name="Wireless Headphones"
  description="Premium noise-cancelling wireless headphones"
  image={{ src: '/products/headphones.jpg', alt: 'Headphones' }}
  price={{ amount: 99, currency: 'USD' }}
  comparePrice={{ amount: 149, currency: 'USD' }}
  brand="AudioTech"
  category="Electronics"
  rating={{ value: 4.5, count: 123 }}
  badge="Sale"
  inStock
  variants={[
    { name: 'Black', value: 'black', image: '/black.jpg' },
    { name: 'White', value: 'white', image: '/white.jpg' },
  ]}
  actions={[
    { label: 'Add to Cart', action: 'addToCart', variant: 'primary' },
    { label: 'Quick View', action: 'quickView', variant: 'outline' },
  ]}
/>
```

## In Product Grid

```mdx
<Section title="Featured Products">
  <Grid cols={4}>
    {products.map(product => (
      <Product key={product.id} {...product} />
    ))}
  </Grid>
</Section>
```

## Detailed Layout

```mdx
<Product
  layout="detailed"
  images={productImages}
  features={['Noise cancelling', '30hr battery', 'Bluetooth 5.0']}
/>
```
