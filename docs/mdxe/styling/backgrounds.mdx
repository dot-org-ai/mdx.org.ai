# Background Effects

Add visual background effects via the `bg` query parameter.

## Available Effects

| Effect | Description |
|--------|-------------|
| `gradient` | Radial gradient overlay |
| `grid` | CSS grid pattern |
| `dots` | Polka dot pattern |
| `waves` | Soft wave gradients |
| `stars` | Star field effect |

## Usage

```html
<link rel="stylesheet" href="/styles/landing.css?bg=gradient&color=purple">
```

Or in frontmatter:

```yaml
---
$type: LandingPage
$style:
  background: gradient
  color: purple
---
```
