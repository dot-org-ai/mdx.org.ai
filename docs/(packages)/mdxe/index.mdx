# mdxe

Execute MDX documents as live, interactive applications across any JavaScript runtime.

## What is mdxe?

mdxe is the execution layer for MDX documents. It compiles and runs MDX content in:

- **Node.js** - Server-side execution
- **Edge** - Cloudflare Workers, Vercel Edge
- **Browser** - Client-side rendering
- **Terminals** - CLI applications with Ink

## Features

- **Universal Execution** - Run MDX anywhere JavaScript runs
- **Isolated Contexts** - Secure sandboxed execution
- **Hot Reloading** - Development with instant feedback
- **Component Injection** - Custom component libraries
- **Streaming** - Progressive rendering support

## Installation

```bash
npm install mdxe
```

Or install runtime-specific packages:

```bash
# Node.js
npm install @mdxe/node

# Next.js
npm install @mdxe/next

# Cloudflare Workers
npm install @mdxe/workers
```

## Quick Example

```typescript
import { execute } from 'mdxe'

const mdx = `
# Hello World

<Counter initial={0} />
`

const result = await execute(mdx, {
  components: {
    Counter: ({ initial }) => <button>{initial}</button>
  }
})
```
