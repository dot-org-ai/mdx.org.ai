# mdxai

Build AI agents and workflows powered by MDX documents.

## What is mdxai?

mdxai connects MDX documents to AI systems, enabling:

- **AI-Powered Content** - Generate and transform MDX with LLMs
- **Agent Workflows** - Define agentic behaviors in MDX
- **Voice Interfaces** - Voice-enabled documentation
- **Multi-Model Support** - Claude, GPT, and more

## Features

- **Semantic Context** - MDXLD types inform AI understanding
- **Tool Integration** - Define AI tools in MDX
- **Streaming** - Real-time content generation
- **Validation** - Type-safe AI outputs

## Integrations

| Package | Platform | Description |
|---------|----------|-------------|

## Installation

```bash
npm install mdxai
```

Or install specific integrations:

```bash
# Claude
npm install @mdxai/claude

# Voice
npm install @mdxai/vapi
```

## Quick Example

```typescript
import { generateMDX } from 'mdxai'

const result = await generateMDX({
  prompt: 'Write a tutorial about React hooks',
  schema: {
    $type: 'Tutorial',
    title: 'string',
    difficulty: 'beginner | intermediate | advanced',
  },
})

console.log(result.frontmatter.title)
console.log(result.content)
```

## Use Cases

### Content Generation

```typescript
import { generateMDX } from '@mdxai/claude'

const blogPost = await generateMDX({
  prompt: 'Write a blog post about TypeScript 5.0 features',
  schema: {
    $type: 'BlogPost',
    title: 'string',
    excerpt: 'string',
    tags: 'string[]',
  },
})
```

### Document Transformation

```typescript
import { transformMDX } from '@mdxai/claude'

const enhanced = await transformMDX(existingMDX, {
  instructions: 'Add code examples and improve explanations',
  preserveStructure: true,
})
```

### Voice Documentation

```typescript
import { createVoiceAgent } from '@mdxai/vapi'

const agent = createVoiceAgent({
  documents: await loadDocs('./docs'),
  voice: 'alloy',
  instructions: 'Help users navigate the documentation',
})
```
