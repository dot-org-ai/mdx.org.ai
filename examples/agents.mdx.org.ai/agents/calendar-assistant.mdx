# Calendar Assistant

The Calendar Assistant agent handles all your scheduling needs using natural language. Tell it "Schedule a meeting with Sarah next week" and it figures out the rest.

## Usage

```typescript
import { CalendarAssistant } from '@agents/calendar-assistant'

const calendar = new CalendarAssistant({
  model: 'claude-3-sonnet',
  calendar: googleCalendar,
  contacts: googleContacts,
})

// Natural language scheduling
await calendar.run('Schedule a 30-minute call with the engineering team tomorrow afternoon')

// Find available slots
const slots = await calendar.findSlots({
  attendees: ['sarah@company.com', 'john@company.com'],
  duration: 60,
  preferredTimes: ['morning'],
})

// Get schedule summary
const summary = await calendar.summarize({ days: 7 })
```

## Configuration

```yaml
# agent.yaml
name: calendar-assistant
model: claude-3-sonnet
tools:
  - calendar_read
  - calendar_write
  - email_send
settings:
  timezone: America/New_York
  workingHours:
    start: "09:00"
    end: "17:00"
  bufferTime: 15  # minutes between meetings
```

## Example Interactions

**User:** "Find time for a team standup next week, 15 minutes, mornings only"

**Agent:** "I found 3 available slots for a 15-minute team standup:
- Monday 9:30 AM - Everyone available
- Wednesday 10:00 AM - Everyone available
- Friday 9:00 AM - Sarah has a conflict, but others available

Would you like me to schedule any of these?"

**User:** "Monday works. Send invites."

**Agent:** "Done! I've scheduled 'Team Standup' for Monday at 9:30 AM and sent calendar invites to all team members."
