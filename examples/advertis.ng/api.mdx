# API Reference

Programmatic access to AI advertising tools.

## Generate Ads

```bash
POST https://advertis.ng/api/generate
Content-Type: application/json

{
  "product": {
    "name": "CloudSync Pro",
    "description": "Real-time file synchronization",
    "price": "$9/mo",
    "url": "https://cloudsync.pro"

  "platform": "meta",
  "objective": "conversions",
  "audience": {
    "interests": ["technology", "productivity"],
    "demographics": { "age": "25-45", "income": "high" }
  },
  "variants": 10
}
```

POST https://advertis.ng/api/experiment
Content-Type: application/json

  "variants": ["var_1", "var_2", "var_3"],
  "platform": "meta",
  "budget": 1000,
  "duration": "7d",
  "metrics": ["ctr", "cpa", "roas"],
  "autoOptimize": true

## SDK

```ts
import { advertis } from 'advertis.ng'

// Generate ads
const ads = await advertis.generate({
  product: { name: 'My SaaS', price: '$29/mo' },
  platform: 'meta',
  variants: 10
})

// Generate images
const images = await advertis.images({
  product: 'My SaaS',
  style: 'modern-minimal',
  variants: 5
})

// Run experiment
const experiment = await advertis.experiment({
  variants: ads.variants,
  budget: 1000,
  duration: '7d'
})

// Get results
const results = await advertis.results(experiment.id)
```

## CLI

```bash
# Generate ads
npx advertis.ng generate --product "My SaaS" --platform meta

# Generate images
npx advertis.ng images --product "My SaaS" --style modern

# Run experiment
npx advertis.ng experiment --variants var_1,var_2 --budget 1000

# Get results
npx advertis.ng results exp_abc123
```
