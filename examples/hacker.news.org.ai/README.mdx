## API

This site is powered by the [Hacker News Firebase API](https://github.com/HackerNews/API).

```ts
import { sources } from '@mdxdb/sources'

const hn = sources.get('https://hacker.news.org.ai')

// Get top stories
const topIds = await hn.endpoint('topStories')

// Get story details
const story = await hn.endpoint('item', { id: '12345' })

// Get user profile
const user = await hn.endpoint('user', { id: 'pg' })
```
