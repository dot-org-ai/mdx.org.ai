# Lean Canvas: builder.domains

## Problem

1. **Domain Cost** - $12+/year per domain adds up for experiments
2. **Commitment Hesitation** - Unsure if project will last, don't want to buy domain
3. **Setup Friction** - DNS, SSL, configuration delays projects

### Existing Alternatives
- GoDaddy/Namecheap (paid per domain)
- Ngrok/Localtunnel (temporary, not professional)
- Vercel/Netlify subdomains (vendor-specific)
- Free TLDs (often spammy/blocked)
