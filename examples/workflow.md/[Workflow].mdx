## Trigger

**Type:** {trigger}

```ts
// Trigger this workflow via:
if (trigger.startsWith('cron')) {
  return <CronDisplay cron={trigger} />
} else if (trigger === 'webhook') {
  return <WebhookUrl url={`https://workflow.md/${$id}/trigger`} />
} else {
  return <EventTrigger event={trigger} />
}
```
