# Interactive MDX Notebook

This is an example notebook demonstrating the interactive features of MDX notebooks.
All code cells have access to `sql`, `db`, `ai`, and `api` globals without importing.

## JavaScript Basics

```javascript
// Simple arithmetic
const x = 10
const y = 20
x + y
```

```javascript

  { id: 3, name: 'Charlie', role: 'user' },

users
```

## Async/Await

```javascript
// Fetch data from an API
const response = await fetch('https://jsonplaceholder.typicode.com/todos/1')
const todo = await response.json()
todo
```

## SQL Queries

Use the `sql` tagged template literal to query remote data:

```javascript
// Query remote JSON data with SQL
const result = await sql`
  SELECT id, title, completed
  FROM url('https://jsonplaceholder.typicode.com/todos')
  WHERE completed = true
  LIMIT 5
`

result
```

## Using the API Helper

```javascript
// The api helper provides convenient methods for HTTP requests
const posts = await api.get('https://jsonplaceholder.typicode.com/posts?_limit=3')
posts
```

## Data Transformation

```javascript
// Transform and analyze data
const todos = await api.get('https://jsonplaceholder.typicode.com/todos')

const stats = {
  total: todos.length,
  completed: todos.filter(t => t.completed).length,
  pending: todos.filter(t => !t.completed).length,
  completionRate: `${((todos.filter(t => t.completed).length / todos.length) * 100).toFixed(1)}%`
}

stats
```

```javascript
// Date manipulation
```
