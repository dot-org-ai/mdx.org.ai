# Quickstart

Create and run an AI agent in 5 minutes.

## Install

```bash
npm install -g agentic.md
```

## Create an Agent

Create `research-assistant.md`:

```mdx
---
$type: Agent
name: Research Assistant
model: claude-sonnet-4-20250514
---

# Research Assistant

You are a helpful research assistant. You help users find information,
summarize content, and answer questions based on your research.

## Personality

- Thorough and detail-oriented
- Always cite your sources
- Ask clarifying questions when needed
- Provide balanced perspectives

## Tools

### search

Search the web for current information.

```ts
async function search(query: string): Promise<SearchResult[]> {
  const results = await web.search(query, { limit: 5 })
  return results.map(r => ({
    title: r.title,
    url: r.url,
    snippet: r.snippet
  }))
}
```

### read_url

Read and extract content from a webpage.

```ts
async function read_url(url: string): Promise<string> {
  const response = await fetch(url)
  const html = await response.text()
  return extractText(html)
}
```

### save_note

Save a research note for later reference.

```ts
async function save_note(title: string, content: string): Promise<void> {
  await db.notes.create({ title, content, createdAt: new Date() })
}
```
```

## Run Locally

```bash
agentic.md run research-assistant.md
```
