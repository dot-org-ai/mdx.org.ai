# API Reference

Programmatic access to the AI-enhanced schema vocabulary.

## Get Type

```bash
GET https://schema.org.ai/api/types/Person
```

```json
{
  "$type": "SchemaType",
  "$id": "https://schema.org.ai/Person",
  "label": "Person",
  "comment": "A person (alive, dead, undead, or fictional).",
  "subClassOf": "Thing",
  "properties": [
    { "name": "name", "expectedType": "Text" },
    { "name": "email", "expectedType": "Text" },
    { "name": "digital", "expectedType": "Agent" }
  ]
}
```

## List Types

```bash
GET https://schema.org.ai/api/types?parent=Thing&limit=20
```

```json
{
  "types": [
    { "name": "Action", "parent": "Thing" },
    { "name": "CreativeWork", "parent": "Thing" },
    { "name": "Event", "parent": "Thing" },
    { "name": "Organization", "parent": "Thing" },
    { "name": "Person", "parent": "Thing" }
  ],
  "total": 850,
  "hasMore": true
}
```

## Get Property

```bash
GET https://schema.org.ai/api/properties/digital
```

```json
{
  "$type": "SchemaProperty",
  "$id": "https://schema.org.ai/digital",
  "name": "digital",
  "description": "The digital twin or AI representation of this entity.",
  "domainIncludes": ["Thing"],
  "rangeIncludes": ["Agent", "DigitalDocument", "URL"],
  "isExtension": true
}
```

## List Properties

```bash
GET https://schema.org.ai/api/properties?domain=Person&limit=20
```

```json
{
  "properties": [
    { "name": "name", "domain": "Thing", "range": "Text" },
    { "name": "email", "domain": "Person", "range": "Text" },
    { "name": "digital", "domain": "Thing", "range": "Agent" }
  ],
  "total": 45,
  "hasMore": true
}
```

## Validate

```bash
POST https://schema.org.ai/api/validate
Content-Type: application/json

{
  "@type": "Person",
  "name": "Alice",
  "email": "alice@example.com",
  "digital": {
    "@type": "Agent",
    "capabilities": ["email"]
  }
}
```

```json
{
  "valid": true,
  "warnings": [],
  "suggestions": [
    { "property": "url", "message": "Consider adding a URL" }
  ]
}
```

## Generate Schema

```bash
POST https://schema.org.ai/api/generate
Content-Type: application/json

{
  "description": "A software company that builds AI tools",
  "context": "structured data for website"
}
```

```json
{
  "@context": "https://schema.org",
  "@type": "Organization",
  "name": "Company Name",
  "description": "A software company that builds AI tools",
  "@digital": {
    "@type": "Agent",
    "capabilities": ["customer-support", "sales"]
  },
  "url": "https://example.com",
  "sameAs": []
}
```

## Search

```bash
GET https://schema.org.ai/api/search?q=agent+capabilities
```

```json
{
  "results": [
    { "type": "Type", "name": "Agent", "score": 0.95 },
    { "type": "Property", "name": "capabilities", "score": 0.92 },
    { "type": "Property", "name": "potentialAction", "score": 0.78 }
  ]
}
```

## SDK

```ts
import { schema } from 'schema.org.ai'

// Get type
const person = await schema.type('Person')

// List properties for a type
const props = await schema.properties({ domain: 'Person' })

// Validate data
const result = await schema.validate({
  '@type': 'Person',
  name: 'Alice'
})

// Generate schema from description
const generated = await schema.generate({
  description: 'A blog post about AI'
})

// Search vocabulary
const results = await schema.search('agent')
```

## CLI

```bash
# Get type info
npx schema.org.ai type Person

# List properties
npx schema.org.ai properties --domain Person

# Validate JSON-LD
npx schema.org.ai validate ./data.json

# Generate schema
npx schema.org.ai generate "A restaurant in San Francisco"

# Search
npx schema.org.ai search "agent"
```
