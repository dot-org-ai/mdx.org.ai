# Stacked Layout

A simple vertical layout for content-focused pages. Header and footer are optional.

<Preview>
  <div data-layout="stacked">
    <header>
      <nav>
        <a href="/">Brand</a>
        <ul>
          <li><a href="#">Features</a></li>
          <li><a href="#">Pricing</a></li>
          <li><a href="#">Docs</a></li>
        </ul>
      </nav>
    </header>
    <main>
      <h1>Main Content Area</h1>
      <p>This is where your page content goes.</p>
    </main>
    <footer>
      <p>&copy; 2024 Brand. All rights reserved.</p>
    </footer>
  </div>
</Preview>

## When to Use

- Marketing sites and landing pages
- Blog posts and articles
- Simple content-focused pages
- Any page that flows vertically

## Structure

```html
<body data-layout="stacked">
  <header>
    <nav>
      <a href="/">Brand</a>
      <ul>
        <li><a href="/features">Features</a></li>
        <li><a href="/pricing">Pricing</a></li>
      </ul>
    </nav>
  </header>

  <main>
    <!-- Page content -->
  </main>

  <footer>
    <!-- Footer content -->
  </footer>
</body>
```

## CSS

```css
[data-layout="stacked"] {
  display: flex;
  flex-direction: column;
  min-height: 100vh;
}

[data-layout="stacked"] > header {
  position: sticky;
  top: 0;
  z-index: 50;
  background: var(--background);
  border-bottom: 1px solid var(--border);
}

[data-layout="stacked"] > header nav {
  display: flex;
  justify-content: space-between;
  align-items: center;
  padding: var(--spacing-4);
  max-width: var(--container-width, 1200px);
  margin: 0 auto;
}

[data-layout="stacked"] > header nav > a {
  font-weight: 700;
}

[data-layout="stacked"] > header ul {
  display: flex;
  gap: var(--spacing-4);
  list-style: none;
}

[data-layout="stacked"] > main {
  flex: 1;
  width: 100%;
  max-width: var(--container-width, 1200px);
  margin: 0 auto;
  padding: var(--spacing-6) var(--spacing-4);
}

[data-layout="stacked"] > footer {
  border-top: 1px solid var(--border);
  padding: var(--spacing-6) var(--spacing-4);
  text-align: center;
  color: var(--muted-foreground);
}
```

## React Native

```tsx
import { SafeAreaView, ScrollView, View } from 'react-native'

export function StackedLayout({ children }) {
  return (
    <SafeAreaView style={{ flex: 1 }}>
      <Header />
      <ScrollView style={{ flex: 1 }}>
        {children}
      </ScrollView>
      <Footer />
    </SafeAreaView>
  )
}
```
