# Stacked Layout

The most common layout pattern: a vertical stack of header, main content, and footer.

<Preview>
  <article>
    <header>
      <nav>
        <a href="/">Brand</a>
        <menu>
          <li><a href="/features">Features</a></li>
          <li><a href="/pricing">Pricing</a></li>
          <li><a href="/docs">Docs</a></li>
        </menu>
      </nav>
    </header>
    <main>
      <h1>Main Content Area</h1>
      <p>This is where your page content goes.</p>
    </main>
    <footer>
      <p>&copy; 2024 Brand. All rights reserved.</p>
    </footer>
  </article>
</Preview>

## Semantic Pattern

A stacked layout is an `<article>` with `<header>`, `<main>`, and `<footer>` as direct children:

```html
<article>
  <header>
    <nav>
      <a href="/">Brand</a>
      <menu>
        <li><a href="/features">Features</a></li>
        <li><a href="/pricing">Pricing</a></li>
      </menu>
    </nav>
  </header>
  <main>
    <!-- Page content -->
  </main>
  <footer>
    <p>&copy; 2024 Brand</p>
  </footer>
</article>
```

## CSS

```css
/* Match article with header + main + footer */
article:has(> header + main + footer) {
  display: flex;
  flex-direction: column;
  min-height: 100vh;
}

article:has(> header + main + footer) > header {
  position: sticky;
  top: 0;
  z-index: 50;
  background: var(--background);
  border-bottom: 1px solid var(--border);
}

article:has(> header + main + footer) > header > nav {
  display: flex;
  justify-content: space-between;
  align-items: center;
  padding: var(--spacing-4);
  max-width: 1200px;
  margin: 0 auto;
}

article:has(> header + main + footer) > header menu {
  display: flex;
  gap: var(--spacing-4);
  list-style: none;
}

article:has(> header + main + footer) > main {
  flex: 1;
  max-width: 1200px;
  margin: 0 auto;
  padding: var(--spacing-6) var(--spacing-4);
  width: 100%;
}

article:has(> header + main + footer) > footer {
  border-top: 1px solid var(--border);
  padding: var(--spacing-6) var(--spacing-4);
  text-align: center;
  color: var(--muted-foreground);
}
```

## When to Use

- Marketing sites and landing pages
- Blog posts and articles
- Simple content-focused pages
- Any page that flows vertically
