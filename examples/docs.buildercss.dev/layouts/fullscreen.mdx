## When to Use

- Video players
- Photo galleries
- Games
- Presentations
- Immersive reading experiences
- Interactive demos

## Structure

```html
<body data-layout="fullscreen">
  <main>
    <!-- Immersive content -->
    <video src="/media/video.mp4" autoplay loop muted></video>
  </main>

  <nav data-controls aria-label="Media controls">
    <button data-action="previous">Previous</button>
    <button data-action="play">Play</button>
    <button data-action="next">Next</button>
  </nav>

  <aside data-overlay>
    <button data-action="settings">Settings</button>
    <button data-action="exit">Exit</button>
  </aside>
</body>
```

## CSS

```css
[data-layout="fullscreen"] {
  position: fixed;
  inset: 0;
  background: black;
  overflow: hidden;
}

[data-layout="fullscreen"] > main {
  width: 100%;
  height: 100%;
  display: flex;
  align-items: center;
  justify-content: center;
}

[data-layout="fullscreen"] > main > video,
[data-layout="fullscreen"] > main > img {
  width: 100%;
  height: 100%;
  object-fit: contain;
}

[data-layout="fullscreen"] [data-controls] {
  position: absolute;
  bottom: var(--spacing-8);
  left: 50%;
  transform: translateX(-50%);
  display: flex;
  gap: var(--spacing-4);
  opacity: 0;
  transition: opacity 0.3s;
}

[data-layout="fullscreen"]:hover [data-controls] {
  opacity: 1;
}

[data-layout="fullscreen"] [data-overlay] {
  position: absolute;
  top: var(--spacing-4);
  right: var(--spacing-4);
  display: flex;
  gap: var(--spacing-2);
  opacity: 0;
  transition: opacity 0.3s;
}

[data-layout="fullscreen"]:hover [data-overlay] {
  opacity: 1;
}
```

## React Native

```tsx
import { View, StatusBar } from 'react-native'

export function FullscreenLayout({ children }) {
  return (
    <View style={styles.container}>
      <StatusBar hidden />
      {children}
    </View>
  )
}

const styles = StyleSheet.create({
  container: {
    flex: 1,
    backgroundColor: 'black',
  },
})
```
