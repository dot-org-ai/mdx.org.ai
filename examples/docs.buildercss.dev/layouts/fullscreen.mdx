# Fullscreen Layout

An edge-to-edge immersive layout for media, games, presentations, and focused experiences.

<Preview>
  <div data-layout="fullscreen">
    <aside>
      <button>Settings</button>
      <button>Exit</button>
    </aside>
    <main>
      <h1>Immersive Experience</h1>
      <p>Full viewport content with minimal UI</p>
    </main>
    <nav aria-label="Media controls">
      <button>&#9664;</button>
      <button aria-current="true">&#9654;</button>
      <button>&#9654;&#9654;</button>
    </nav>
  </div>
</Preview>

## When to Use

- Video players
- Photo galleries
- Games
- Presentations
- Immersive reading experiences
- Interactive demos

## Structure

```html
<body data-layout="fullscreen">
  <aside>
    <button>Settings</button>
    <button>Exit</button>
  </aside>

  <main>
    <!-- Immersive content -->
    <video src="/media/video.mp4" autoplay loop muted></video>
  </main>

  <nav aria-label="Media controls">
    <button>Previous</button>
    <button>Play</button>
    <button>Next</button>
  </nav>
</body>
```

## CSS

```css
[data-layout="fullscreen"] {
  position: fixed;
  inset: 0;
  background: linear-gradient(135deg, #1a1a2e 0%, #16213e 100%);
  overflow: hidden;
  display: flex;
  align-items: center;
  justify-content: center;
}

[data-layout="fullscreen"] > main {
  text-align: center;
  color: white;
}

[data-layout="fullscreen"] > main h1 {
  font-size: var(--text-4xl);
  font-weight: 700;
  margin-bottom: var(--spacing-4);
}

[data-layout="fullscreen"] > main p {
  color: rgba(255, 255, 255, 0.7);
}

[data-layout="fullscreen"] > main > video,
[data-layout="fullscreen"] > main > img {
  width: 100%;
  height: 100%;
  object-fit: contain;
}

[data-layout="fullscreen"] > aside {
  position: absolute;
  top: var(--spacing-4);
  right: var(--spacing-4);
  display: flex;
  gap: var(--spacing-2);
  opacity: 0;
  transition: opacity 0.3s;
}

[data-layout="fullscreen"]:hover > aside {
  opacity: 1;
}

[data-layout="fullscreen"] > aside button {
  padding: var(--spacing-2);
  background: rgba(255, 255, 255, 0.1);
  border: none;
  border-radius: var(--radius);
  color: white;
  cursor: pointer;
}

[data-layout="fullscreen"] > nav {
  position: absolute;
  bottom: var(--spacing-4);
  left: 50%;
  transform: translateX(-50%);
  display: flex;
  gap: var(--spacing-4);
  opacity: 0;
  transition: opacity 0.3s;
}

[data-layout="fullscreen"]:hover > nav {
  opacity: 1;
}

[data-layout="fullscreen"] > nav button {
  width: 48px;
  height: 48px;
  border-radius: 50%;
  background: rgba(255, 255, 255, 0.2);
  border: none;
  color: white;
  cursor: pointer;
}

[data-layout="fullscreen"] > nav button[aria-current] {
  background: white;
  color: black;
}
```

## React Native

```tsx
import { View, StatusBar } from 'react-native'

export function FullscreenLayout({ children }) {
  return (
    <View style={styles.container}>
      <StatusBar hidden />
      {children}
    </View>
  )
}

const styles = StyleSheet.create({
  container: {
    flex: 1,
    backgroundColor: 'black',
  },
})
```
