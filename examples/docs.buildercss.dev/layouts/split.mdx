# Split Layout

Two equal-width content panels, useful for comparisons, editors, or side-by-side views.

<Preview>
  <article>
    <main>
      <header><h2>Editor</h2></header>
      <pre><code>const hello = "world"</code></pre>
    </main>
    <main>
      <header><h2>Preview</h2></header>
      <p>Hello, world!</p>
    </main>
  </article>
</Preview>

## Semantic Pattern

A split layout is an `<article>` with two `<main>` elements as siblings:

```html
<article>
  <main>
    <header><h2>Left Panel</h2></header>
    <!-- Left content -->
  </main>
  <main>
    <header><h2>Right Panel</h2></header>
    <!-- Right content -->
  </main>
</article>
```

## CSS

```css
/* Match article with two main elements */
article:has(> main + main) {
  display: grid;
  grid-template-columns: 1fr 1fr;
  min-height: 100vh;
}

article:has(> main + main) > main {
  overflow-y: auto;
}

article:has(> main + main) > main:first-child {
  border-right: 1px solid var(--border);
}

article:has(> main + main) > main > header {
  padding: var(--spacing-3) var(--spacing-4);
  border-bottom: 1px solid var(--border);
  background: var(--muted);
}

article:has(> main + main) > main > header h2 {
  font-size: var(--text-sm);
  font-weight: 600;
}

@media (max-width: 768px) {
  article:has(> main + main) {
    grid-template-columns: 1fr;
    grid-template-rows: 1fr 1fr;
  }

  article:has(> main + main) > main:first-child {
    border-right: none;
    border-bottom: 1px solid var(--border);
  }
}
```

## When to Use

- Code editors with preview
- Diff viewers
- Translation interfaces
- Before/after comparisons
- Chat with sidebar
