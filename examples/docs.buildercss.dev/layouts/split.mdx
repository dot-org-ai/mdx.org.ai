# Split Layout

A two-panel layout perfect for onboarding flows, auth screens, and focused content presentation.

<Preview>
  <div data-layout="split">
    <aside>
      <figure>Visual Content</figure>
    </aside>
    <main>
      <h1>Welcome aboard</h1>
      <p>Get started with our platform in just a few steps.</p>
      <button>Get Started</button>
    </main>
  </div>
</Preview>

## When to Use

- Onboarding flows
- Authentication screens
- Feature introductions
- Landing page sections

## Structure

```html
<div data-layout="split">
  <aside>
    <img src="/hero.jpg" alt="" />
  </aside>
  <main>
    <h1>Welcome aboard</h1>
    <p>Get started with our platform in just a few steps.</p>
    <button>Get Started</button>
  </main>
</div>
```

## CSS

```css
[data-layout="split"] {
  display: grid;
  grid-template-columns: 1fr 1fr;
  min-height: 100vh;
}

[data-layout="split"] > aside {
  display: flex;
  align-items: center;
  justify-content: center;
  background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
  padding: var(--spacing-8);
}

[data-layout="split"] > aside img {
  max-width: 100%;
  max-height: 100%;
  object-fit: contain;
}

[data-layout="split"] > main {
  display: flex;
  flex-direction: column;
  align-items: center;
  justify-content: center;
  padding: var(--spacing-8);
  max-width: 400px;
  margin: auto;
}

[data-layout="split"] h1 {
  font-size: var(--text-3xl);
  font-weight: 700;
  margin-bottom: var(--spacing-4);
}

[data-layout="split"] p {
  color: var(--muted-foreground);
  margin-bottom: var(--spacing-6);
}

[data-layout="split"] button {
  padding: var(--spacing-2) var(--spacing-4);
  background: var(--primary);
  color: var(--primary-foreground);
  border: none;
  border-radius: var(--radius);
  cursor: pointer;
}

@media (max-width: 768px) {
  [data-layout="split"] {
    grid-template-columns: 1fr;
  }

  [data-layout="split"] > aside {
    min-height: 200px;
  }
}
```

## Variations

### Reversed

```html
<div data-layout="split" dir="rtl">
  <main dir="ltr">...</main>
  <aside>...</aside>
</div>
```

### With Form

```html
<div data-layout="split">
  <aside>
    <img src="/brand.svg" alt="" />
  </aside>
  <main>
    <h1>Create account</h1>
    <form>
      <label>Email</label>
      <input type="email" />
      <label>Password</label>
      <input type="password" />
      <button type="submit">Sign Up</button>
    </form>
  </main>
</div>
```
