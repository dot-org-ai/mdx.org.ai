# Split Layout

A 50/50 split layout ideal for comparison views, onboarding, and hero sections with visual content.

## When to Use

- Onboarding flows
- Comparison views (before/after)
- Hero sections with imagery
- Login/signup with branding
- Feature highlights

## Structure

```html
<body data-layout="split">
  <section data-panel="visual">
    <img src="/hero-image.jpg" alt="Hero" />
  </section>

  <section data-panel="content">
    <article>
      <h1>Welcome aboard</h1>
      <p>Get started with our platform in just a few steps.</p>
      <a href="/signup" data-button="primary">Get Started</a>
    </article>
  </section>
</body>
```

## CSS

```css
[data-layout="split"] {
  display: grid;
  grid-template-columns: 1fr 1fr;
  min-height: 100vh;
}

[data-layout="split"] > [data-panel="visual"] {
  background: var(--primary);
  display: flex;
  align-items: center;
  justify-content: center;
  overflow: hidden;
}

[data-layout="split"] > [data-panel="visual"] > img {
  width: 100%;
  height: 100%;
  object-fit: cover;
}

[data-layout="split"] > [data-panel="content"] {
  display: flex;
  align-items: center;
  justify-content: center;
  padding: var(--spacing-8);
}

[data-layout="split"] > [data-panel="content"] > article {
  max-width: 400px;
}

@media (max-width: 768px) {
  [data-layout="split"] {
    grid-template-columns: 1fr;
    grid-template-rows: 1fr 1fr;
  }

  [data-layout="split"] > [data-panel="visual"] {
    max-height: 40vh;
  }
}
```

## Variations

### Reversed Split

```html
<body data-layout="split" data-reverse>
  <section data-panel="content">...</section>
  <section data-panel="visual">...</section>
</body>
```

```css
[data-layout="split"][data-reverse] {
  direction: rtl;
}

[data-layout="split"][data-reverse] > * {
  direction: ltr;
}
```

### Asymmetric Split

```css
[data-layout="split"][data-ratio="40-60"] {
  grid-template-columns: 40% 60%;
}

[data-layout="split"][data-ratio="30-70"] {
  grid-template-columns: 30% 70%;
}
```

## React Native

```tsx
import { View } from 'react-native'

export function SplitLayout({ left, right }) {
  return (
    <View style={styles.container}>
      <View style={styles.panel}>{left}</View>
      <View style={styles.panel}>{right}</View>
    </View>
  )
}

const styles = StyleSheet.create({
  container: {
    flex: 1,
    flexDirection: 'row',
  },
  panel: {
    flex: 1,
    justifyContent: 'center',
    alignItems: 'center',
  },
})
```
