# Sidebar Layout

A two-column layout with a persistent sidebar for navigation. Best for applications and dashboards.

<Preview>
  <div data-layout="sidebar" style={{ display: 'grid', gridTemplateColumns: '200px 1fr', minHeight: '400px', border: '1px solid var(--border)', borderRadius: 'var(--radius-lg)', overflow: 'hidden' }}>
    <aside style={{ background: 'var(--muted)', padding: 'var(--spacing-4)', borderRight: '1px solid var(--border)' }}>
      <nav>
        <h3 style={{ fontSize: 'var(--text-sm)', fontWeight: 600, marginBottom: 'var(--spacing-2)' }}>Navigation</h3>
        <ul style={{ listStyle: 'none', padding: 0, display: 'flex', flexDirection: 'column', gap: 'var(--spacing-1)' }}>
          <li><a href="#" style={{ display: 'block', padding: 'var(--spacing-2)', borderRadius: 'var(--radius)', background: 'var(--primary)', color: 'var(--primary-foreground)' }}>Dashboard</a></li>
          <li><a href="#" style={{ display: 'block', padding: 'var(--spacing-2)' }}>Projects</a></li>
          <li><a href="#" style={{ display: 'block', padding: 'var(--spacing-2)' }}>Settings</a></li>
        </ul>
      </nav>
    </aside>
    <main style={{ padding: 'var(--spacing-6)' }}>
      <h1>Dashboard</h1>
      <p>Main content area with full width.</p>
    </main>
  </div>
</Preview>

## When to Use

- Application dashboards
- Admin panels
- Settings pages
- Any page with persistent navigation

## Structure

```html
<body data-layout="sidebar">
  <aside data-sidebar>
    <nav>
      <a href="/" data-logo>Brand</a>
      <ul>
        <li><a href="/dashboard" aria-current="page">Dashboard</a></li>
        <li><a href="/projects">Projects</a></li>
        <li><a href="/settings">Settings</a></li>
      </ul>
    </nav>
  </aside>

  <main>
    <!-- Page content -->
  </main>
</body>
```

## CSS

```css
[data-layout="sidebar"] {
  display: grid;
  grid-template-columns: 280px 1fr;
  min-height: 100vh;
}

[data-layout="sidebar"] > [data-sidebar] {
  background: var(--muted);
  border-right: 1px solid var(--border);
  padding: var(--spacing-4);
  position: sticky;
  top: 0;
  height: 100vh;
  overflow-y: auto;
}

[data-layout="sidebar"] > main {
  padding: var(--spacing-6);
  overflow-y: auto;
}

@media (max-width: 768px) {
  [data-layout="sidebar"] {
    grid-template-columns: 1fr;
  }

  [data-layout="sidebar"] > [data-sidebar] {
    display: none;
  }
}
```

## React Native

```tsx
import { View, ScrollView } from 'react-native'

export function SidebarLayout({ children, sidebar }) {
  return (
    <View style={{ flex: 1, flexDirection: 'row' }}>
      <View style={styles.sidebar}>
        {sidebar}
      </View>
      <ScrollView style={{ flex: 1 }}>
        {children}
      </ScrollView>
    </View>
  )
}
```
