# Sidebar Layout

A two-column layout with a persistent sidebar for navigation. Best for applications and dashboards.

<Preview>
  <article>
    <aside>
      <nav>
        <a href="/">Brand</a>
        <menu>
          <li><a href="/dashboard" aria-current="page">Dashboard</a></li>
          <li><a href="/projects">Projects</a></li>
          <li><a href="/settings">Settings</a></li>
        </menu>
      </nav>
    </aside>
    <main>
      <header>
        <h1>Dashboard</h1>
      </header>
      <p>Main content area with full width.</p>
    </main>
  </article>
</Preview>

## Semantic Pattern

A sidebar layout is an `<article>` with an `<aside>` containing `<nav>` as first child:

```html
<article>
  <aside>
    <nav>
      <a href="/">Brand</a>
      <menu>
        <li><a href="/dashboard" aria-current="page">Dashboard</a></li>
        <li><a href="/projects">Projects</a></li>
        <li><a href="/settings">Settings</a></li>
      </menu>
    </nav>
  </aside>
  <main>
    <!-- Page content -->
  </main>
</article>
```

## CSS

```css
/* Match article with aside > nav as first child */
article:has(> aside:first-child > nav) {
  display: grid;
  grid-template-columns: 280px 1fr;
  min-height: 100vh;
}

article:has(> aside:first-child > nav) > aside {
  background: var(--muted);
  border-right: 1px solid var(--border);
  position: sticky;
  top: 0;
  height: 100vh;
  overflow-y: auto;
}

article:has(> aside:first-child > nav) > aside > nav {
  padding: var(--spacing-4);
}

article:has(> aside:first-child > nav) > aside menu {
  list-style: none;
  padding: 0;
  display: flex;
  flex-direction: column;
  gap: var(--spacing-1);
}

article:has(> aside:first-child > nav) > aside a {
  display: block;
  padding: var(--spacing-2) var(--spacing-3);
  border-radius: var(--radius);
  text-decoration: none;
  color: inherit;
}

article:has(> aside:first-child > nav) > aside a[aria-current] {
  background: var(--primary);
  color: var(--primary-foreground);
}

article:has(> aside:first-child > nav) > main {
  padding: var(--spacing-6);
  overflow-y: auto;
}

@media (max-width: 768px) {
  article:has(> aside:first-child > nav) {
    grid-template-columns: 1fr;
  }

  article:has(> aside:first-child > nav) > aside {
    display: none;
  }
}
```

## When to Use

- Application dashboards
- Admin panels
- Settings pages
- Any page with persistent navigation
