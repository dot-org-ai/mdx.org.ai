# Sidebar Layout

A two-column layout with a persistent sidebar for navigation. Best for applications and dashboards.

<Preview>
  <div data-layout="sidebar">
    <aside>
      <nav>
        <h3>Navigation</h3>
        <ul>
          <li><a href="#" aria-current="page">Dashboard</a></li>
          <li><a href="#">Projects</a></li>
          <li><a href="#">Settings</a></li>
        </ul>
      </nav>
    </aside>
    <main>
      <h1>Dashboard</h1>
      <p>Main content area with full width.</p>
    </main>
  </div>
</Preview>

## When to Use

- Application dashboards
- Admin panels
- Settings pages
- Any page with persistent navigation

## Structure

```html
<body data-layout="sidebar">
  <aside>
    <nav>
      <a href="/">Brand</a>
      <ul>
        <li><a href="/dashboard" aria-current="page">Dashboard</a></li>
        <li><a href="/projects">Projects</a></li>
        <li><a href="/settings">Settings</a></li>
      </ul>
    </nav>
  </aside>

  <main>
    <!-- Page content -->
  </main>
</body>
```

## CSS

```css
[data-layout="sidebar"] {
  display: grid;
  grid-template-columns: 280px 1fr;
  min-height: 100vh;
}

[data-layout="sidebar"] > aside {
  background: var(--muted);
  border-right: 1px solid var(--border);
  padding: var(--spacing-4);
  position: sticky;
  top: 0;
  height: 100vh;
  overflow-y: auto;
}

[data-layout="sidebar"] > aside h3 {
  font-size: var(--text-sm);
  font-weight: 600;
  margin-bottom: var(--spacing-2);
}

[data-layout="sidebar"] > aside ul {
  list-style: none;
  padding: 0;
  display: flex;
  flex-direction: column;
  gap: var(--spacing-1);
}

[data-layout="sidebar"] > aside a {
  display: block;
  padding: var(--spacing-2);
  border-radius: var(--radius);
}

[data-layout="sidebar"] > aside a[aria-current] {
  background: var(--primary);
  color: var(--primary-foreground);
}

[data-layout="sidebar"] > main {
  padding: var(--spacing-6);
  overflow-y: auto;
}

@media (max-width: 768px) {
  [data-layout="sidebar"] {
    grid-template-columns: 1fr;
  }

  [data-layout="sidebar"] > aside {
    display: none;
  }
}
```

## React Native

```tsx
import { View, ScrollView } from 'react-native'

export function SidebarLayout({ children, sidebar }) {
  return (
    <View style={{ flex: 1, flexDirection: 'row' }}>
      <View style={styles.sidebar}>
        {sidebar}
      </View>
      <ScrollView style={{ flex: 1 }}>
        {children}
      </ScrollView>
    </View>
  )
}
```
