## When to Use

- Documentation sites
- Technical guides
- API references
- Knowledge bases

## Structure

```html
<body data-layout="docs">
  <header>
    <nav><!-- Top navigation --></nav>
  </header>

  <aside data-sidebar>
    <nav>
      <section>
        <h4>Getting Started</h4>
        <ul>
          <li><a href="/docs/intro">Introduction</a></li>
          <li><a href="/docs/install" aria-current="page">Installation</a></li>
        </ul>
      </section>
    </nav>
  </aside>

  <main>
    <article>
      <!-- Documentation content -->
    </article>
  </main>

  <aside data-toc>
    <nav aria-label="Table of contents">
      <h4>On this page</h4>
      <ul>
        <li><a href="#npm">Using npm</a></li>
        <li><a href="#cdn">Using CDN</a></li>
      </ul>
    </nav>
  </aside>
</body>
```

## CSS

```css
[data-layout="docs"] {
  display: grid;
  grid-template-columns: 280px 1fr 200px;
  grid-template-rows: auto 1fr;
  min-height: 100vh;
}

[data-layout="docs"] > header {
  grid-column: 1 / -1;
  border-bottom: 1px solid var(--border);
}

[data-layout="docs"] > [data-sidebar] {
  padding: var(--spacing-4);
  border-right: 1px solid var(--border);
  position: sticky;
  top: 60px;
  height: calc(100vh - 60px);
  overflow-y: auto;
}

[data-layout="docs"] > main {
  padding: var(--spacing-8);
  max-width: 800px;
}

[data-layout="docs"] > [data-toc] {
  padding: var(--spacing-4);
  position: sticky;
  top: 60px;
  height: calc(100vh - 60px);
}

@media (max-width: 1024px) {
  [data-layout="docs"] {
    grid-template-columns: 1fr;
  }

  [data-layout="docs"] > [data-sidebar],
  [data-layout="docs"] > [data-toc] {
    display: none;
  }
}
```
