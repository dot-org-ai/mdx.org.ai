# Docs Layout

A three-column layout optimized for documentation with a sidebar navigation and table of contents.

<Preview>
  <div data-layout="docs">
    <aside>
      <nav>
        <section>
          <h4>Getting Started</h4>
          <ul>
            <li><a href="#">Introduction</a></li>
            <li><a href="#" aria-current="page">Installation</a></li>
            <li><a href="#">Quick Start</a></li>
          </ul>
        </section>
      </nav>
    </aside>
    <main>
      <article>
        <h1>Installation</h1>
        <p>Learn how to install and configure BuilderCSS.</p>
        <h2 id="npm">Using npm</h2>
        <p>Install via package manager:</p>
        <pre>npm install buildercss</pre>
      </article>
    </main>
    <aside>
      <nav aria-label="Table of contents">
        <h4>On this page</h4>
        <ul>
          <li><a href="#npm">Using npm</a></li>
          <li><a href="#cdn">Using CDN</a></li>
        </ul>
      </nav>
    </aside>
  </div>
</Preview>

## When to Use

- Documentation sites
- Technical guides
- API references
- Knowledge bases

## Structure

```html
<body data-layout="docs">
  <header>
    <nav><!-- Top navigation --></nav>
  </header>

  <aside>
    <nav>
      <section>
        <h4>Getting Started</h4>
        <ul>
          <li><a href="/docs/intro">Introduction</a></li>
          <li><a href="/docs/install" aria-current="page">Installation</a></li>
        </ul>
      </section>
    </nav>
  </aside>

  <main>
    <article>
      <!-- Documentation content -->
    </article>
  </main>

  <aside>
    <nav aria-label="Table of contents">
      <h4>On this page</h4>
      <ul>
        <li><a href="#npm">Using npm</a></li>
        <li><a href="#cdn">Using CDN</a></li>
      </ul>
    </nav>
  </aside>
</body>
```

## CSS

```css
[data-layout="docs"] {
  display: grid;
  grid-template-columns: 280px 1fr 200px;
  grid-template-rows: auto 1fr;
  min-height: 100vh;
}

[data-layout="docs"] > header {
  grid-column: 1 / -1;
  border-bottom: 1px solid var(--border);
}

[data-layout="docs"] > aside:first-of-type {
  padding: var(--spacing-4);
  border-right: 1px solid var(--border);
  position: sticky;
  top: 60px;
  height: calc(100vh - 60px);
  overflow-y: auto;
}

[data-layout="docs"] > aside h4 {
  font-size: var(--text-xs);
  font-weight: 600;
  text-transform: uppercase;
  margin-bottom: var(--spacing-2);
}

[data-layout="docs"] > aside ul {
  list-style: none;
  padding: 0;
  font-size: var(--text-sm);
}

[data-layout="docs"] > aside a[aria-current] {
  color: var(--primary);
}

[data-layout="docs"] > main {
  padding: var(--spacing-8);
  max-width: 800px;
}

[data-layout="docs"] > main pre {
  background: var(--muted);
  padding: var(--spacing-4);
  border-radius: var(--radius);
}

[data-layout="docs"] > aside:last-of-type {
  padding: var(--spacing-4);
  border-left: 1px solid var(--border);
  position: sticky;
  top: 60px;
  height: calc(100vh - 60px);
}

@media (max-width: 1024px) {
  [data-layout="docs"] {
    grid-template-columns: 1fr;
  }

  [data-layout="docs"] > aside {
    display: none;
  }
}
```
