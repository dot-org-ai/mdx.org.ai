## When to Use

- SaaS applications
- Admin dashboards
- Internal tools
- Complex web applications

## Structure

```html
<body data-layout="dashboard">
  <aside data-sidebar>
    <header>
      <a href="/" data-logo>Acme Inc</a>
    </header>
    <nav>
      <ul>
        <li><a href="/dashboard" aria-current="page">Dashboard</a></li>
        <li><a href="/projects">Projects</a></li>
        <li><a href="/team">Team</a></li>
        <li><a href="/settings">Settings</a></li>
      </ul>
    </nav>
  </aside>

  <div data-main>
    <header>
      <input type="search" placeholder="Search..." />
      <div data-user-menu>
        <img src="/avatar.jpg" alt="User" />
      </div>
    </header>

    <main>
      <!-- Views go here -->
    </main>
  </div>
</body>
```

## CSS

```css
[data-layout="dashboard"] {
  display: grid;
  grid-template-columns: 280px 1fr;
  min-height: 100vh;
}

[data-layout="dashboard"] > [data-sidebar] {
  background: var(--sidebar-background, var(--foreground));
  color: var(--sidebar-foreground, var(--background));
  display: flex;
  flex-direction: column;
}

[data-layout="dashboard"] [data-main] {
  display: flex;
  flex-direction: column;
}

[data-layout="dashboard"] [data-main] > header {
  display: flex;
  align-items: center;
  justify-content: space-between;
  padding: var(--spacing-4);
  border-bottom: 1px solid var(--border);
}

[data-layout="dashboard"] [data-main] > main {
  flex: 1;
  padding: var(--spacing-6);
  overflow-y: auto;
}

@media (max-width: 768px) {
  [data-layout="dashboard"] {
    grid-template-columns: 1fr;
  }

  [data-layout="dashboard"] > [data-sidebar] {
    position: fixed;
    inset: 0;
    z-index: 50;
    transform: translateX(-100%);
    transition: transform 0.3s;
  }

  [data-layout="dashboard"][data-sidebar-open] > [data-sidebar] {
    transform: translateX(0);
  }
}
```
