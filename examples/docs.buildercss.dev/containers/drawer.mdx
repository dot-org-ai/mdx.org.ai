# Drawer

A bottom sheet that slides up from the bottom of the screen, commonly used on mobile.

## Semantic Pattern

A drawer uses `<aside aria-modal role="dialog">` to indicate a modal dialog from the bottom. The header contains a drag handle (`<span>`) for touch gestures:

```html
<aside aria-modal="true" role="dialog" aria-hidden="true">
  <article>
    <header>
      <span></span><!-- Drag handle -->
      <h2>Filter Results</h2>
    </header>
    <section>
      <!-- Filter options -->
    </section>
    <footer>
      <menu>
        <li><button>Reset</button></li>
        <li><button>Apply Filters</button></li>
      </menu>
    </footer>
  </article>
</aside>
```

## JavaScript (with Gesture Support)

```js
const drawer = document.querySelector('aside[aria-modal][role="dialog"]')
const article = drawer.querySelector('article')
const handle = drawer.querySelector('header > span:first-child')

let startY = 0
let currentY = 0
let isDragging = false

handle.addEventListener('touchstart', (e) => {
  isDragging = true
  startY = e.touches[0].clientY
})

document.addEventListener('touchmove', (e) => {
  if (!isDragging) return
  currentY = e.touches[0].clientY - startY
  if (currentY > 0) {
    article.style.transform = `translateY(${currentY}px)`
  }
})

document.addEventListener('touchend', () => {
  if (!isDragging) return
  isDragging = false

  if (currentY > 100) {
    drawer.setAttribute('aria-hidden', 'true')
  }

  article.style.transform = ''
  currentY = 0
})

// Close on backdrop click
drawer.addEventListener('click', (e) => {
  if (e.target === drawer) {
    drawer.setAttribute('aria-hidden', 'true')
  }
})
```

## When to Use

- Mobile filters
- Action sheets
- Share menus
- Quick options
- Mobile navigation
