## When to Use

- Mobile filters
- Action sheets
- Share menus
- Quick options
- Mobile navigation

## Structure

```html
<aside data-drawer aria-modal="true" aria-hidden="true">
  <article>
    <header>
      <span data-handle></span>
      <h2>Filter Results</h2>
    </header>
    <section>
      <!-- Filter options -->
    </section>
    <footer>
      <button data-button="ghost">Reset</button>
      <button data-button="primary">Apply Filters</button>
    </footer>
  </article>
</aside>
```

## JavaScript (with Gesture Support)

```js
const drawer = document.querySelector('[data-drawer]')
const article = drawer.querySelector('article')
const handle = drawer.querySelector('[data-handle]')

let startY = 0
let currentY = 0
let isDragging = false

handle.addEventListener('touchstart', (e) => {
  isDragging = true
  startY = e.touches[0].clientY
})

document.addEventListener('touchmove', (e) => {
  if (!isDragging) return
  currentY = e.touches[0].clientY - startY
  if (currentY > 0) {
    article.style.transform = `translateY(${currentY}px)`
  }
})

document.addEventListener('touchend', () => {
  if (!isDragging) return
  isDragging = false

  if (currentY > 100) {
    drawer.setAttribute('aria-hidden', 'true')
  }

  article.style.transform = ''
  currentY = 0
})

// Close on backdrop click
drawer.addEventListener('click', (e) => {
  if (e.target === drawer) {
    drawer.setAttribute('aria-hidden', 'true')
  }
})
```

## Variations

### Action Sheet

```html
<aside data-drawer data-variant="actions">
  <article>
    <header>
      <span data-handle></span>
    </header>
    <nav>
      <button data-action="share">Share</button>
      <button data-action="copy">Copy Link</button>
      <button data-action="edit">Edit</button>
      <button data-action="delete" data-destructive>Delete</button>
    </nav>
    <footer>
      <button data-action="close">Cancel</button>
    </footer>
  </article>
</aside>
```

### Map/Full Height

```html
<aside data-drawer data-height="full">
  <article>
    <!-- Can be dragged to different snap points -->
  </article>
</aside>
```
