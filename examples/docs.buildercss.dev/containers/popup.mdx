# Popup

A small floating widget in the corner, commonly used for chat interfaces.

## When to Use

- Live chat widgets
- Help/support bubbles
- Quick feedback forms
- Notification centers

## Structure

```html
<!-- Trigger button -->
<button aria-label="Open chat">
  <svg><!-- Chat icon --></svg>
</button>

<!-- Popup -->
<aside data-container="popup" aria-hidden="true">
  <article>
    <header>
      <h2>Chat with us</h2>
      <button aria-label="Close">&times;</button>
    </header>
    <section>
      <!-- Chat messages -->
    </section>
    <footer>
      <input type="text" placeholder="Type a message..." />
      <button type="submit">Send</button>
    </footer>
  </article>
</aside>
```

## JavaScript

```js
const trigger = document.querySelector('[data-container="popup"] > button:first-child')
const popup = document.querySelector('[data-container="popup"]')

trigger.addEventListener('click', () => {
  const isHidden = popup.getAttribute('aria-hidden') === 'true'
  popup.setAttribute('aria-hidden', !isHidden)
})
```
