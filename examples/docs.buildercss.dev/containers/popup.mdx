# Popup

A small floating widget in the corner, commonly used for chat interfaces.

## Semantic Pattern

A popup widget uses `<aside aria-label>` with a trigger button. The pattern `aside:has(> button[aria-expanded] + article)` identifies it:

```html
<!-- Popup Widget -->
<aside aria-label="Chat widget">
  <button aria-label="Open chat" aria-expanded="false">
    <svg><!-- Chat icon --></svg>
  </button>
  <article aria-hidden="true">
    <header>
      <h2>Chat with us</h2>
      <button aria-label="Close">&times;</button>
    </header>
    <section>
      <!-- Chat messages -->
    </section>
    <footer>
      <input type="text" placeholder="Type a message..." />
      <button type="submit">Send</button>
    </footer>
  </article>
</aside>
```

## JavaScript

```js
const widget = document.querySelector('aside[aria-label="Chat widget"]')
const trigger = widget.querySelector('button[aria-expanded]')

trigger.addEventListener('click', () => {
  const isExpanded = trigger.getAttribute('aria-expanded') === 'true'
  trigger.setAttribute('aria-expanded', !isExpanded)
})
```

## When to Use

- Live chat widgets
- Help/support bubbles
- Quick feedback forms
- Notification centers
