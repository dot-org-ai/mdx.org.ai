## When to Use

- Live chat widgets
- Help/support bubbles
- Quick feedback forms
- Notification centers

## Structure

```html
<!-- Trigger button -->
<button data-popup-trigger aria-label="Open chat">
  <svg><!-- Chat icon --></svg>
</button>

<!-- Popup -->
<aside data-popup aria-hidden="true">
  <article>
    <header>
      <h2>Chat with us</h2>
      <button data-action="close">&times;</button>
    </header>
    <section data-messages>
      <!-- Chat messages -->
    </section>
    <footer>
      <input type="text" placeholder="Type a message..." />
      <button data-button="primary">Send</button>
    </footer>
  </article>
</aside>
```

## JavaScript

```js
const trigger = document.querySelector('[data-popup-trigger]')
const popup = document.querySelector('[data-popup]')

trigger.addEventListener('click', () => {
  const isHidden = popup.getAttribute('aria-hidden') === 'true'
  popup.setAttribute('aria-hidden', !isHidden)

  // Animate trigger icon
  trigger.classList.toggle('active')
})

// Close button
popup.querySelector('[data-action="close"]').addEventListener('click', () => {
  popup.setAttribute('aria-hidden', 'true')
  trigger.classList.remove('active')
})
```

```html

  <article>
    <header>
      <h2>Notifications</h2>
      <button data-action="mark-read">Mark all read</button>
    </header>
    <ul data-notification-list>
      <li data-notification data-unread>
```
