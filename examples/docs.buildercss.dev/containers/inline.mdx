# Inline Container

Content embedded directly within the page flow, always visible without overlays.

<Preview>
  <div>
    <p>Regular page content flows here...</p>

    <article data-container="inline">
      <header>
        <span>NEW</span>
        <h3>Introducing Dark Mode</h3>
      </header>
      <p>Experience our app in a whole new way with our brand new dark mode feature.</p>
      <nav>
        <button>Try it now</button>
        <a href="#">Learn more</a>
      </nav>
    </article>

    <p>More content continues below...</p>
  </div>
</Preview>

## When to Use

- Feature announcements
- Inline CTAs
- Embedded widgets
- Contextual cards
- Tip/note boxes
- Inline forms

## Structure

```html
<article>
  <p>Regular content...</p>

  <aside data-container="inline">
    <header>
      <span>NEW</span>
      <h3>Feature Announcement</h3>
    </header>
    <p>Description of the feature...</p>
    <nav>
      <a href="/try">Try it now</a>
      <a href="/learn">Learn more</a>
    </nav>
  </aside>

  <p>More content...</p>
</article>
```

### Dismissible Banner

```html
<aside data-container="inline">
  <p>üéâ New feature available! Check out our latest update.</p>
  <button aria-label="Dismiss">&times;</button>
</aside>
```

```js
document.querySelectorAll('[data-container="inline"] button[aria-label="Dismiss"]').forEach(btn => {
  btn.addEventListener('click', () => {
    const container = btn.closest('[data-container="inline"]')
    container.remove()
    // Optionally persist dismissal
    localStorage.setItem(`dismissed-${container.id}`, 'true')
  })
})
```
