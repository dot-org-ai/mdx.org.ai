## When to Use

- Feature announcements
- Inline CTAs
- Embedded widgets
- Contextual cards
- Tip/note boxes
- Inline forms

## Structure

```html
<article>
  <p>Regular content...</p>

  <aside data-container="inline">
    <header>
      <span data-badge>NEW</span>
      <h3>Feature Announcement</h3>
    </header>
    <p>Description of the feature...</p>
    <div data-actions>
      <a href="/try" data-button="primary">Try it now</a>
      <a href="/learn" data-button="outline">Learn more</a>
    </div>
  </aside>

  <p>More content...</p>
</article>
```

## CSS

```css
[data-container="inline"] {
  background: var(--muted);
  border: 1px solid var(--border);
  border-radius: var(--radius-lg);
  padding: var(--spacing-6);
  margin: var(--spacing-6) 0;
}

[data-container="inline"] > header {
  margin-bottom: var(--spacing-4);
}

[data-container="inline"] > header [data-badge] {
  display: inline-block;
  padding: var(--spacing-1) var(--spacing-2);
  background: var(--primary);
  color: var(--primary-foreground);
  border-radius: var(--radius);
  font-size: var(--text-xs);
  margin-bottom: var(--spacing-2);

  margin-bottom: var(--spacing-4);

[data-container="inline"] [data-actions] {
  display: flex;
  gap: var(--spacing-2);
}
```

```html

  <header>
    <h3>Subscribe to updates</h3>
  </header>
  <form>
    <input type="email" placeholder="Enter your email" />
    <button type="submit" data-button="primary">Subscribe</button>
  </form>

```

### Dismissible Banner

```html
<aside data-container="inline" data-dismissible>
  <p>üéâ New feature available! Check out our latest update.</p>
  <button data-action="dismiss" aria-label="Dismiss">&times;</button>
</aside>
```

```js
document.querySelectorAll('[data-dismissible] [data-action="dismiss"]').forEach(btn => {
  btn.addEventListener('click', () => {
    const container = btn.closest('[data-container="inline"]')
    container.remove()
    // Optionally persist dismissal
    localStorage.setItem(`dismissed-${container.id}`, 'true')
  })
})
```
