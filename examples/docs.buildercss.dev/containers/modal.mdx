## When to Use

- Confirmation dialogs
- Quick forms
- Alerts and notifications
- Focused interactions

## Structure

```html
<!-- Trigger -->
<button data-modal-trigger="confirm-delete">Delete</button>

<!-- Modal -->
<aside data-modal id="confirm-delete" aria-hidden="true">
  <article>
    <header>
      <h2>Delete Item?</h2>
      <button data-action="close" aria-label="Close">&times;</button>
    </header>
    <section>
      <p>This action cannot be undone. Are you sure?</p>
    </section>
    <footer>
      <button data-button="ghost" data-action="close">Cancel</button>
      <button data-button="destructive">Delete</button>
    </footer>
  </article>
</aside>
```

## CSS

```css
[data-modal] {
  position: fixed;
  inset: 0;
  z-index: 100;
  display: flex;
  align-items: center;
  justify-content: center;
}

[data-modal]::before {
  content: '';
  position: absolute;
  inset: 0;
  background: rgba(0, 0, 0, 0.5);
}

[data-modal] > article {
  position: relative;
  background: var(--background);
  border-radius: var(--radius-lg);
  max-width: 500px;
  width: 100%;
  margin: var(--spacing-4);
  animation: modalIn 0.2s ease;
}

[data-modal] > article > header {
  display: flex;
  justify-content: space-between;
  align-items: center;
  padding: var(--spacing-4);
  border-bottom: 1px solid var(--border);
}

[data-modal] > article > section {
  padding: var(--spacing-4);
}

[data-modal] > article > footer {
  display: flex;
  justify-content: flex-end;
  gap: var(--spacing-2);
  padding: var(--spacing-4);
  border-top: 1px solid var(--border);
}

[data-modal][aria-hidden="true"] {
  display: none;
}

@keyframes modalIn {
  from {
    opacity: 0;
    transform: scale(0.95);
  }
  to {
    opacity: 1;
    transform: scale(1);
  }
}
```

## JavaScript

```js
// Open modal
document.querySelectorAll('[data-modal-trigger]').forEach(trigger => {
  trigger.addEventListener('click', () => {
    const modalId = trigger.dataset.modalTrigger
    const modal = document.getElementById(modalId)
    modal.setAttribute('aria-hidden', 'false')
  })
})

// Close modal
document.querySelectorAll('[data-modal] [data-action="close"]').forEach(btn => {
  btn.addEventListener('click', () => {
    const modal = btn.closest('[data-modal]')
    modal.setAttribute('aria-hidden', 'true')
  })
})

// Close on backdrop click
document.querySelectorAll('[data-modal]').forEach(modal => {
  modal.addEventListener('click', (e) => {
    if (e.target === modal) {
      modal.setAttribute('aria-hidden', 'true')
    }
  })
})

// Close on Escape
document.addEventListener('keydown', (e) => {
  if (e.key === 'Escape') {
    document.querySelectorAll('[data-modal][aria-hidden="false"]').forEach(modal => {
      modal.setAttribute('aria-hidden', 'true')
    })
  }
})
```

## Variations

### Form Modal

```html
<aside data-modal id="edit-profile">
  <article>
    <header>
      <h2>Edit Profile</h2>
      <button data-action="close">&times;</button>
    </header>
    <form>
      <section>
        <div data-field>
          <label for="name">Name</label>
          <input type="text" id="name" />
        </div>
      </section>
      <footer>
        <button type="button" data-button="ghost" data-action="close">Cancel</button>
        <button type="submit" data-button="primary">Save</button>
      </footer>
    </form>
  </article>
</aside>
```

### Alert Modal

```html
<aside data-modal data-variant="alert">
  <article>
    <section>
      <div data-icon="warning">⚠️</div>
      <h2>Session Expired</h2>
      <p>Please log in again to continue.</p>
    </section>
    <footer>
      <button data-button="primary" data-action="close">Log In</button>
    </footer>
  </article>
</aside>
```
