# Modal Container

A centered dialog overlay for confirmations, forms, and focused content.

<Preview>
  <div data-container="modal">
    <article>
      <header>
        <h2>Confirm Action</h2>
        <button aria-label="Close">×</button>
      </header>
      <p>Are you sure you want to continue? This action cannot be undone.</p>
      <footer>
        <button>Cancel</button>
        <button>Confirm</button>
      </footer>
    </article>
  </div>
</Preview>

## When to Use

- Confirmation dialogs
- Form submissions
- Alert messages
- Focused tasks

## Structure

```html
<div data-container="modal">
  <article>
    <header>
      <h2>Confirm Action</h2>
      <button aria-label="Close">×</button>
    </header>
    <p>Are you sure you want to continue?</p>
    <footer>
      <button>Cancel</button>
      <button>Confirm</button>
    </footer>
  </article>
</div>
```

## JavaScript

```js
const modal = document.querySelector('[data-container="modal"]')

// Close on backdrop click
modal.addEventListener('click', (e) => {
  if (e.target === modal) {
    modal.hidden = true
  }
})

// Close on escape key
document.addEventListener('keydown', (e) => {
  if (e.key === 'Escape' && !modal.hidden) {
    modal.hidden = true
  }
})

// Open modal
function openModal() {
  modal.hidden = false
  modal.querySelector('article').focus()
}
```

## Accessibility

- Use `role="dialog"` and `aria-modal="true"` for screen readers
- Trap focus within the modal when open
- Return focus to trigger element on close
- Support Escape key to dismiss

## Variations

### Destructive

```html
<div data-container="modal">
  <article>
    <header>
      <h2>Delete Item</h2>
      <button aria-label="Close">×</button>
    </header>
    <p>This action cannot be undone.</p>
    <footer>
      <button>Cancel</button>
      <button>Delete</button>
    </footer>
  </article>
</div>
```

```css
[data-container="modal"] footer button:last-child:has(+ :empty) {
  /* When it's destructive action */
  background: var(--destructive);
  color: var(--destructive-foreground);
}
```

### With Form

```html
<div data-container="modal">
  <article>
    <header>
      <h2>Edit Profile</h2>
      <button aria-label="Close">×</button>
    </header>
    <form>
      <label>Name</label>
      <input type="text" />
      <label>Email</label>
      <input type="email" />
      <footer>
        <button type="button">Cancel</button>
        <button type="submit">Save</button>
      </footer>
    </form>
  </article>
</div>
```
