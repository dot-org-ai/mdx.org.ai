# Slideover

A side panel that slides in from the left or right edge of the screen.

<Preview>
  <div data-container="slideover">
    <article>
      <header>
        <h2>Settings</h2>
        <button aria-label="Close">&times;</button>
      </header>
      <section>
        <div>
          <label>Email notifications</label>
          <p>Receive email updates about your account.</p>
        </div>
        <div>
          <label>Theme</label>
          <select>
            <option>System</option>
            <option>Light</option>
            <option>Dark</option>
          </select>
        </div>
      </section>
      <footer>
        <button>Cancel</button>
        <button>Save</button>
      </footer>
    </article>
  </div>
</Preview>

## When to Use

- Settings panels
- Detail views
- Filters and facets
- Secondary navigation
- Cart sidebars

## Structure

```html
<!-- Trigger -->
<button>Settings</button>

<!-- Slideover -->
<aside data-container="slideover" aria-hidden="true">
  <article>
    <header>
      <h2>Settings</h2>
      <button aria-label="Close">&times;</button>
    </header>
    <section>
      <!-- Settings content -->
    </section>
    <footer>
      <button>Cancel</button>
      <button>Save Changes</button>
    </footer>
  </article>
</aside>
```

## JavaScript

```js
// Open slideover
document.querySelectorAll('[data-slideover-trigger]').forEach(trigger => {
  trigger.addEventListener('click', () => {
    const slideover = document.querySelector('[data-container="slideover"]')
    slideover.setAttribute('aria-hidden', 'false')
  })
})

// Close on button click
document.querySelectorAll('[data-container="slideover"] header button').forEach(btn => {
  btn.addEventListener('click', () => {
    const slideover = btn.closest('[data-container="slideover"]')
    slideover.setAttribute('aria-hidden', 'true')
  })
})
```
