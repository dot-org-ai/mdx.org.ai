# Slideover

A side panel that slides in from the left or right edge of the screen.

<Preview>
  <aside aria-modal="true">
    <article>
      <header>

      <footer>
        <menu>
          <li><button>Cancel</button></li>

      </footer>
    </article>
  </aside>

## Semantic Pattern

A slideover uses `<aside aria-modal="true">` containing an article with settings/panel content. The pattern identifies a side panel:

```html
<!-- Trigger -->
<button>Settings</button>

<!-- Slideover -->
<aside aria-modal="true" aria-hidden="true">
  <article>
    <header>
      <h2>Settings</h2>
      <button aria-label="Close">&times;</button>
    </header>
    <section>
      <!-- Settings content -->
    </section>
    <footer>
      <menu>
        <li><button>Cancel</button></li>
        <li><button>Save Changes</button></li>
      </menu>
    </footer>
  </article>
</aside>
```

## JavaScript

```js
// Open slideover
document.querySelectorAll('[aria-controls]').forEach(trigger => {
  trigger.addEventListener('click', () => {
    const slideover = document.getElementById(trigger.getAttribute('aria-controls'))
    slideover.setAttribute('aria-hidden', 'false')
  })
})

// Close on button click
document.querySelectorAll('aside[aria-modal] header button').forEach(btn => {
  btn.addEventListener('click', () => {
    const slideover = btn.closest('aside[aria-modal]')
    slideover.setAttribute('aria-hidden', 'true')
  })
})
```

## When to Use

- Settings panels
- Detail views
- Filters and facets
- Secondary navigation
- Cart sidebars
