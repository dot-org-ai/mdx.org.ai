# CTA Section

A call-to-action section to drive conversions.

<Preview>
  <section>
    <h2>Ready to get started?</h2>
    <p>Join thousands of happy customers today.</p>
    <menu>
      <li><a href="/signup">Start Free Trial</a></li>
      <li><a href="/contact">Talk to Sales</a></li>
    </menu>
  </section>
</Preview>

## Semantic Pattern

A CTA section uses `<menu>` for action links. The pattern `section:has(> h2 + p + menu)` identifies it:

```html
<section>
  <h2>Ready to get started?</h2>
  <p>Join thousands of happy customers today.</p>
  <menu>
    <li><a href="/signup">Start Free Trial</a></li>
    <li><a href="/contact">Talk to Sales</a></li>
  </menu>
</section>
```

## CSS

```css
/* CTA: section with h2 + p + menu */
section:has(> h2 + p + menu) {
  padding: var(--spacing-16) var(--spacing-4);
  background: var(--primary);
  color: var(--primary-foreground);
  text-align: center;
  border-radius: var(--radius-lg);
}

section:has(> h2 + p + menu) h2 {
  font-size: var(--text-3xl);
  font-weight: 700;
  margin-bottom: var(--spacing-4);
}

section:has(> h2 + p + menu) > p {
  margin-bottom: var(--spacing-8);
  opacity: 0.9;
}

section:has(> h2 + p + menu) menu {
  display: flex;
  gap: var(--spacing-4);
  justify-content: center;
  list-style: none;
}

/* Primary action: first link */
section:has(> h2 + p + menu) menu > :first-child a {
  background: var(--background);
  color: var(--foreground);
  padding: var(--spacing-3) var(--spacing-6);
  border: none;
  border-radius: var(--radius);
  font-weight: 500;
  text-decoration: none;
}

/* Secondary action: other links */
section:has(> h2 + p + menu) menu > :not(:first-child) a {
  padding: var(--spacing-3) var(--spacing-6);
  background: transparent;
  border: 1px solid currentColor;
  border-radius: var(--radius);
  font-weight: 500;
  color: inherit;
  text-decoration: none;
}

@media (max-width: 640px) {
  section:has(> h2 + p + menu) menu {
    flex-direction: column;
    align-items: center;
  }
}
```

## Variations

### Newsletter CTA

Use a form for email signup:

```html
<section>
  <h2>Stay in the loop</h2>
  <p>Get updates on new features and releases.</p>
  <form>
    <input type="email" placeholder="Enter your email" required />
    <button type="submit">Subscribe</button>
  </form>
  <small>No spam, unsubscribe anytime.</small>
</section>
```

### Minimal CTA

```html
<section>
  <p>Questions? <a href="/contact">Contact us</a></p>
</section>
```

## When to Use

- End of landing pages
- Newsletter signups
- Trial conversions
- Contact prompts
