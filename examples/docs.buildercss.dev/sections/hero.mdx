# Hero Section

The primary landing section that captures attention with a headline, description, and call-to-action.

<Preview>
  <section data-section="hero">
    <h1>Build better products with semantic HTML</h1>
    <p>A design system that uses data attributes instead of utility classes. Clean markup, powerful styling.</p>
    <nav>
      <button>Get Started</button>
      <a href="#">View Documentation</a>
    </nav>
  </section>
</Preview>

## When to Use

- Homepage primary section
- Product landing pages
- Campaign pages
- App marketing sites

## Structure

```html
<section data-section="hero">
  <h1>Build better products with semantic HTML</h1>
  <p>A design system that uses data attributes instead of utility classes.</p>
  <nav>
    <button>Get Started</button>
    <a href="/docs">View Documentation</a>
  </nav>
</section>
```

## CSS

```css
[data-section="hero"] {
  padding: var(--spacing-16) var(--spacing-6);
  text-align: center;
}

[data-section="hero"] h1 {
  font-size: var(--text-4xl);
  font-weight: 800;
  margin-bottom: var(--spacing-4);
  line-height: 1.1;
}

[data-section="hero"] p {
  font-size: var(--text-lg);
  color: var(--muted-foreground);
  max-width: 600px;
  margin: 0 auto var(--spacing-6);
}

[data-section="hero"] nav {
  display: flex;
  gap: var(--spacing-3);
  justify-content: center;
}

[data-section="hero"] button,
[data-section="hero"] a {
  padding: var(--spacing-3) var(--spacing-6);
  border-radius: var(--radius);
  font-weight: 500;
  text-decoration: none;
  cursor: pointer;
}

[data-section="hero"] nav > :first-child {
  background: var(--primary);
  color: var(--primary-foreground);
  border: none;
}

[data-section="hero"] nav > :not(:first-child) {
  background: transparent;
  border: 1px solid var(--border);
  color: var(--foreground);
}
```
