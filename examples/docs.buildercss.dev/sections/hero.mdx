# Hero Section

The primary landing section that captures attention with a headline, description, and call-to-action.

<Preview>
  <section>
    <h1>Build better products with semantic HTML</h1>
    <p>A design system that uses pure HTML structure for styling. Clean markup, powerful CSS.</p>
    <menu>
      <li><a href="/docs">Get Started</a></li>
      <li><a href="/demo">View Demo</a></li>
    </menu>
  </section>
</Preview>

## Semantic Pattern

A hero section is a `<section>` with `<h1>` as first child:

```html
<section>
  <h1>Build better products with semantic HTML</h1>
  <p>A design system that uses pure HTML structure for styling.</p>
  <menu>
    <li><a href="/docs">Get Started</a></li>
    <li><a href="/demo">View Demo</a></li>
  </menu>
</section>
```

## CSS

```css
/* Hero: section with h1 as first child */
section:has(> h1:first-child) {
  padding: var(--spacing-16) var(--spacing-6);
  text-align: center;
}

section:has(> h1:first-child) > h1 {
  font-size: var(--text-4xl);
  font-weight: 800;
  margin-bottom: var(--spacing-4);
  line-height: 1.1;
}

section:has(> h1:first-child) > p {
  font-size: var(--text-lg);
  color: var(--muted-foreground);
  max-width: 600px;
  margin: 0 auto var(--spacing-6);
}

section:has(> h1:first-child) > menu {
  display: flex;
  gap: var(--spacing-3);
  justify-content: center;
  list-style: none;
  padding: 0;
}

section:has(> h1:first-child) > menu a {
  padding: var(--spacing-3) var(--spacing-6);
  border-radius: var(--radius);
  font-weight: 500;
  text-decoration: none;
}

/* Primary action: first menu item */
section:has(> h1:first-child) > menu > :first-child a {
  background: var(--primary);
  color: var(--primary-foreground);
}

/* Secondary action: not first */
section:has(> h1:first-child) > menu > :not(:first-child) a {
  background: transparent;
  border: 1px solid var(--border);
  color: var(--foreground);
}
```

## Variations

### With Badge

Add a `<mark>` element before the heading:

```html
<section>
  <mark>New: Version 2.0 Released</mark>
  <h1>Build better products</h1>
  <p>...</p>
  <menu>...</menu>
</section>
```

```css
section:has(> h1) > mark:first-child {
  display: inline-block;
  padding: var(--spacing-1) var(--spacing-3);
  background: var(--primary);
  color: var(--primary-foreground);
  border-radius: var(--radius-full);
  font-size: var(--text-sm);
  margin-bottom: var(--spacing-4);
}
```

### With Social Proof

Add a `<footer>` with avatars:

```html
<section>
  <h1>...</h1>
  <p>...</p>
  <menu>...</menu>
  <footer>
    <figure>
      <img src="/avatar1.jpg" alt="">
      <img src="/avatar2.jpg" alt="">
      <img src="/avatar3.jpg" alt="">
    </figure>
    <span>Join 10,000+ developers</span>
  </footer>
</section>
```

## When to Use

- Homepage primary section
- Product landing pages
- Campaign pages
- App marketing sites
