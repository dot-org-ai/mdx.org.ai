## Semantic Pattern

A logos section is a `<section>` containing a row of company logos. The pattern `section:has(> p + div > figure)` identifies it:

```html
<section>
  <p>Trusted by teams at</p>
  <div>
    <img src="/logos/company1.svg" alt="Company 1" />
    <img src="/logos/company2.svg" alt="Company 2" />
    <img src="/logos/company3.svg" alt="Company 3" />
    <img src="/logos/company4.svg" alt="Company 4" />
    <img src="/logos/company5.svg" alt="Company 5" />
  </div>
</section>
```

## CSS

```css
/* Logos: section with p + div containing images/figures */
section:has(> p + div > figure),
section:has(> p + div > img) {
  padding: var(--spacing-12) var(--spacing-4);
  text-align: center;
}

section:has(> p + div > figure) > p,
section:has(> p + div > img) > p {
  color: var(--muted-foreground);
  font-size: var(--text-sm);
  margin-bottom: var(--spacing-8);
}

section:has(> p + div > figure) > div,
section:has(> p + div > img) > div {
  display: flex;
  justify-content: center;
  align-items: center;
  gap: var(--spacing-12);
  flex-wrap: wrap;
}

section:has(> p + div > img) > div img {
  height: 32px;
  width: auto;
  opacity: 0.6;
  filter: grayscale(100%);
  transition: opacity 0.2s, filter 0.2s;
}

section:has(> p + div > img) > div img:hover {
  opacity: 1;
  filter: grayscale(0%);
}

section:has(> p + div > figure) > div figure {
  width: 120px;
  height: 40px;
  background: var(--muted);
  border-radius: var(--radius);
  display: flex;
  align-items: center;
  justify-content: center;
  opacity: 0.6;
  margin: 0;
}

@media (max-width: 768px) {
  section:has(> p + div > figure) > div,
  section:has(> p + div > img) > div {
    gap: var(--spacing-6);
  }

  section:has(> p + div > img) > div img {
    height: 24px;
  }
}
```

## Variations

### Scrolling Marquee

```html
<section>
  <p>Trusted by teams at</p>
  <div>
    <div>
      <img src="/logos/company1.svg" alt="" />
      <img src="/logos/company2.svg" alt="" />
      <!-- Duplicate for seamless loop -->
    </div>
  </div>
</section>
```

```css
section:has(> p + div > div > img) > div {
  overflow: hidden;
  mask-image: linear-gradient(
    to right,
    transparent,
    black 20%,
    black 80%,
    transparent
  );
}

section:has(> p + div > div > img) > div > div {
  display: flex;
  gap: var(--spacing-12);
  animation: marquee 30s linear infinite;
}

@keyframes marquee {
  from { transform: translateX(0); }
  to { transform: translateX(-50%); }
}
```

### With Links

```html
<section>
  <div>
    <a href="https://company1.com">
      <img src="/logos/company1.svg" alt="Company 1" />
    </a>
    <!-- More logos -->
  </div>
</section>
```
