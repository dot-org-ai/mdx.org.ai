- Team introductions
- Company culture
- Leadership showcase

## CSS

```css
[data-section="team"] {
  padding: var(--spacing-16) var(--spacing-4);
}

[data-section="team"] > header {
  text-align: center;
  margin-bottom: var(--spacing-12);
}

[data-section="team"] [data-team-grid] {
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(200px, 1fr));
  gap: var(--spacing-8);
  max-width: 900px;
  margin: 0 auto;
}

[data-section="team"] [data-team-member] {
  text-align: center;
}

[data-section="team"] [data-team-member] img {
  width: 120px;
  height: 120px;
  border-radius: 50%;
  object-fit: cover;
  margin: 0 auto var(--spacing-4);
}

[data-section="team"] [data-team-member] h3 {
  font-weight: 600;
  margin-bottom: var(--spacing-1);
}

[data-section="team"] [data-team-member] > p {
  color: var(--muted-foreground);
  font-size: var(--text-sm);
  margin-bottom: var(--spacing-2);
}

[data-section="team"] [data-team-member] nav {
  display: flex;
  justify-content: center;
  gap: var(--spacing-2);
}

[data-section="team"] [data-team-member] nav a {
  color: var(--muted-foreground);
  transition: color 0.2s;
}

[data-section="team"] [data-team-member] nav a:hover {
  color: var(--foreground);
}
```
