- About pages
- Team introductions
- Company culture
- Leadership showcase

## CSS

```css
[data-section="team"] {
  padding: var(--spacing-16) var(--spacing-4);
}

[data-section="team"] > header {
  text-align: center;
  margin-bottom: var(--spacing-12);
}

[data-section="team"] > header h2 {
  font-size: var(--text-3xl);
  font-weight: 700;
  margin-bottom: var(--spacing-4);
}

[data-section="team"] > header p {
  color: var(--muted-foreground);
}

[data-section="team"] > div {
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(200px, 1fr));
  gap: var(--spacing-8);
  max-width: 900px;
  margin: 0 auto;
}

[data-section="team"] article {
  text-align: center;
}

[data-section="team"] article img,
[data-section="team"] article figure {
  width: 120px;
  height: 120px;
  border-radius: 50%;
  object-fit: cover;
  margin: 0 auto var(--spacing-4);
  background: var(--muted);
}

[data-section="team"] article h3 {
  font-weight: 600;
  margin-bottom: var(--spacing-1);
}

[data-section="team"] article > p {
  color: var(--muted-foreground);
  font-size: var(--text-sm);
  margin-bottom: var(--spacing-2);
}

[data-section="team"] article nav {
  display: flex;
  justify-content: center;
  gap: var(--spacing-2);
}

[data-section="team"] article nav a {
  color: var(--muted-foreground);
  transition: color 0.2s;
}

[data-section="team"] article nav a:hover {
  color: var(--foreground);
}
```
