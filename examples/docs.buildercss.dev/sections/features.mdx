## Semantic Pattern

A features section is a `<section>` containing a list of `<article>` elements:

```html
<section>
  <header>
    <mark>Features</mark>
    <h2>Everything you need to build</h2>
    <p>Powerful features to help you ship faster.</p>
  </header>
  <ul>
    <li>
      <article>
        <h3>Feature Name</h3>
        <p>Feature description.</p>
      </article>
    </li>
    <li>
      <article>
        <h3>Feature Name</h3>
        <p>Feature description.</p>
      </article>
    </li>
  </ul>
</section>
```
