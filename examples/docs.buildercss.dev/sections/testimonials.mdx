## Semantic Pattern

A testimonials section uses `<blockquote>` elements inside a container:

```html
<section>
  <header>
    <h2>Loved by developers</h2>
    <p>See what our customers have to say.</p>
  </header>

  <div>
    <blockquote>
      <p>"BuilderCSS transformed how we build UIs."</p>
      <footer>
        <img src="/avatar.jpg" alt="" />
        <div>
          <cite>Jane Doe</cite>
          <span>CTO at Acme Inc</span>
        </div>
      </footer>
    </blockquote>

    <blockquote>
      <p>"Finally, a CSS framework that works with how we think."</p>
      <footer>
        <img src="/avatar2.jpg" alt="" />
        <div>
          <cite>John Smith</cite>
          <span>Lead Developer at StartupCo</span>
        </div>
      </footer>
    </blockquote>
  </div>
</section>
```

```css
  padding: var(--spacing-16) var(--spacing-4);

  text-align: center;
  margin-bottom: var(--spacing-12);
}

section:has(> div > blockquote) > header h2 {
  font-size: var(--text-3xl);
  font-weight: 700;
  margin-bottom: var(--spacing-4);
}

section:has(> div > blockquote) > header p {
  color: var(--muted-foreground);
}

  display: grid;

  margin-bottom: var(--spacing-4);
  font-style: italic;
}

section:has(> div > blockquote) blockquote > footer {
  display: flex;

  color: var(--muted-foreground);

```

## Variations

### With Rating Stars

```html
<blockquote>
  <div>★★★★★</div>
  <p>"Amazing product!"</p>
  <footer>...</footer>
</blockquote>
```

### Featured Testimonial

Use `aria-current="true"` for featured quotes:

```html
<section>
  <blockquote aria-current="true">
    <p>"This completely changed how we work..."</p>
    <footer>
      <img src="/avatar.jpg" alt="" />
      <div>
        <cite>CEO Name</cite>
        <span>Fortune 500 Company</span>
      </div>
    </footer>
  </blockquote>
</section>
```

## When to Use

- Social proof sections
- Customer stories
- Trust building
- Case study highlights
