## JavaScript

```js
document.querySelectorAll('[data-accordion-item] button').forEach(button => {
  button.addEventListener('click', () => {
    const expanded = button.getAttribute('aria-expanded') === 'true'
    button.setAttribute('aria-expanded', !expanded)
  })
})
```
