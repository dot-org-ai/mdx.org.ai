# Sections

Sections are reusable content blocks that compose together to build pages. They're the building blocks of marketing sites and landing pages.

## Section Structure

All sections follow a consistent pattern:

```html
<section data-section="[type]">
  <header>
    <span data-badge>Optional eyebrow</span>
    <h2>Section Heading</h2>
    <p>Section description text.</p>
  </header>

  <div data-content>
    <!-- Section-specific content -->
  </div>

  <footer>
    <!-- Optional actions -->
  </footer>
</section>
```

## Composing Sections

Pages are built by stacking sections:

```html
<main>
  <section data-section="hero">...</section>
  <section data-section="logos">...</section>
  <section data-section="features">...</section>
  <section data-section="testimonials">...</section>
  <section data-section="stats">...</section>
  <section data-section="faq">...</section>
  <section data-section="cta">...</section>
</main>
```

The order and combination of sections creates different page experiences.
