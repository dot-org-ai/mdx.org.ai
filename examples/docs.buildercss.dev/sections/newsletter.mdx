# Newsletter Section

An email signup form for newsletter subscriptions.

## Semantic Pattern

A newsletter section contains a heading and an email form. The pattern `section:has(> h2 + p + form)` identifies it:

```html
<section>
  <h2>Stay up to date</h2>
  <p>Get the latest updates delivered to your inbox.</p>

  <form>
    <input
      type="email"
      placeholder="Enter your email"
      required
      aria-label="Email address"
    />
    <button type="submit">Subscribe</button>
  </form>

  <small>No spam. Unsubscribe anytime.</small>
</section>
```

## JavaScript

```js
const form = document.querySelector('section:has(> h2 + p + form) form')

form.addEventListener('submit', async (e) => {
  e.preventDefault()

  const email = form.querySelector('input[type="email"]').value
  const button = form.querySelector('button[type="submit"]')

  button.disabled = true
  button.textContent = 'Subscribing...'

  try {
    await fetch('/api/subscribe', {
      method: 'POST',
      headers: { 'Content-Type': 'application/json' },
      body: JSON.stringify({ email })
    })

    button.textContent = 'Subscribed!'
    form.reset()
  } catch (error) {
    button.textContent = 'Try again'
  } finally {
    button.disabled = false
  }
})
```
