## Semantic Pattern

A stats section uses the `<data>` element with a machine-readable `value` attribute:

```html
<section>
  <div>
    <article>
      <data value="10000">10K+</data>
      <p>Active Users</p>
    </article>

    <article>
      <data value="99.9">99.9%</data>
      <p>Uptime</p>
    </article>

    <article>
      <data value="50000000">50M+</data>
      <p>API Requests</p>
    </article>

    <article>
      <data value="24">24/7</data>
      <p>Support</p>
    </article>
  </div>
</section>
```

## CSS

```css
/* Stats: section with articles containing data elements */
section:has(> div > article > data) {
  padding: var(--spacing-16) var(--spacing-4);
}

section:has(> div > article > data) > div {
  display: grid;
  grid-template-columns: repeat(4, 1fr);
  gap: var(--spacing-8);
  text-align: center;
  max-width: 900px;
  margin: 0 auto;
}

section:has(> div > article > data) article data {
  display: block;
  font-size: var(--text-4xl);
  font-weight: 700;
  color: var(--primary);
  margin-bottom: var(--spacing-2);
}

section:has(> div > article > data) article p {
  color: var(--muted-foreground);
}

@media (max-width: 768px) {
  section:has(> div > article > data) > div {
    grid-template-columns: repeat(2, 1fr);
  }
}

@media (max-width: 480px) {
  section:has(> div > article > data) > div {
    grid-template-columns: 1fr;
  }
}
```

## Variations

### With Icons

```html
<section>
  <article>
    <figure>
      <svg><!-- Users icon --></svg>
    </figure>
    <data value="10000">10K+</data>
    <p>Active Users</p>
  </article>
</section>
```

### Animated Counters

```html
<section>
  <article>
    <data value="10000">0</data>
    <p>Active Users</p>
  </article>
</section>
```

```js
// Counter animation
const counters = document.querySelectorAll('section:has(> div > article > data) data')
const observer = new IntersectionObserver(entries => {
  entries.forEach(entry => {
    if (entry.isIntersecting) {
      const counter = entry.target
      const target = parseInt(counter.getAttribute('value'))
      animateCounter(counter, target)
    }
  })
})

counters.forEach(counter => observer.observe(counter))
```

### With Dark Background

Use `aria-current="true"` for inverted styles:

```html
<section aria-current="true">
  <div>
    <!-- Stats with inverted colors -->
  </div>
</section>
```

```css
section:has(> div > article > data)[aria-current] {
  background: var(--foreground);
  color: var(--background);
}

section:has(> div > article > data)[aria-current] article data {
  color: var(--background);
}
```
