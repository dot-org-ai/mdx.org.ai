## Semantic Pattern

Terms of service uses `<article>` with header, important notice aside, and content sections. The pattern `article:has(> header > h1 + p > time):has(> aside + div > section)` identifies terms pages:

```html
<article>
  <header>
    <h1>Terms of Service</h1>
    <p><time datetime="2024-12-01">Effective date: December 1, 2024</time></p>
  </header>

  <aside>
    <p><strong>Important:</strong> Please read these terms carefully...</p>
  </aside>

  <div>
    <section>
      <h2>1. Acceptance of Terms</h2>
      <p>By accessing and using Builder CSS...</p>
    </section>

    <section>
      <h2>2. Use License</h2>
      <p>Permission is granted to temporarily use...</p>
      <ul>
        <li>Modify or copy the materials</li>
        <li>Use the materials for any commercial purpose</li>
      </ul>
    </section>

    <section>
      <h2>3. Disclaimer</h2>
      <p>The materials on Builder CSS are provided...</p>
    </section>
  </div>
</article>
```
