## Semantic Pattern

A contact page uses `<main>` with a header, form, and aside with contact details. The pattern `main:has(> div > form + aside > dl)` identifies contact pages:

```html
<main>
  <header>
    <h1>Get in Touch</h1>
    <p>Have questions? We'd love to hear from you.</p>
  </header>

  <div>
    <form>
      <fieldset>
        <div>
          <label for="firstName">First Name</label>
          <input type="text" id="firstName" placeholder="John" />
        </div>
        <div>
          <label for="lastName">Last Name</label>
          <input type="text" id="lastName" placeholder="Doe" />
        </div>
      </fieldset>
      <div>
        <label for="email">Email</label>
        <input type="email" id="email" placeholder="john@example.com" />
      </div>
      <div>
        <label for="subject">Subject</label>
        <select id="subject">
          <option>General Inquiry</option>
          <option>Sales</option>
        </select>
      </div>
      <div>
        <label for="message">Message</label>
        <textarea id="message" rows="4" placeholder="How can we help?"></textarea>
      </div>
      <button type="submit">Send Message</button>
    </form>

    <aside>
      <dl>
        <div>
          <dt>Email</dt>
          <dd>hello@example.com</dd>
        </div>
        <div>
          <dt>Phone</dt>
          <dd>+1 (555) 123-4567</dd>
        </div>
        <div>
          <dt>Address</dt>
          <dd>123 Main Street<br />San Francisco, CA 94102</dd>
        </div>
      </dl>
    </aside>
  </div>
</main>
```

## CSS

```css
/* Contact page: main with form + aside containing contact info */
main:has(> div > form + aside > dl) {
  padding: var(--spacing-6);
}

main:has(> div > form + aside > dl) > header {
  text-align: center;
  max-width: 600px;
  margin: 0 auto var(--spacing-8);
}

main:has(> div > form + aside > dl) > header h1 {
  font-size: var(--text-3xl);
  font-weight: 700;
  margin-bottom: var(--spacing-3);
}

main:has(> div > form + aside > dl) > header p {
  color: var(--muted-foreground);
}

main:has(> div > form + aside > dl) > div {
  display: grid;
  grid-template-columns: 1fr 1fr;
  gap: var(--spacing-8);
  max-width: 900px;
  margin: 0 auto;
}

main:has(> div > form + aside > dl) form {
  display: flex;
  flex-direction: column;
  gap: var(--spacing-4);
}

main:has(> div > form + aside > dl) form fieldset {
  display: grid;
  grid-template-columns: 1fr 1fr;
  gap: var(--spacing-4);
  border: none;
  padding: 0;
  margin: 0;
}

main:has(> div > form + aside > dl) form label {
  display: block;
  font-weight: 500;
  margin-bottom: var(--spacing-1);
  font-size: var(--text-sm);
}

main:has(> div > form + aside > dl) form input,
main:has(> div > form + aside > dl) form select,
main:has(> div > form + aside > dl) form textarea {
  width: 100%;
  padding: var(--spacing-2);
  border: 1px solid var(--border);
  border-radius: var(--radius);
}

main:has(> div > form + aside > dl) form textarea {
  resize: vertical;
}

main:has(> div > form + aside > dl) form button {
  padding: var(--spacing-3);
  background: var(--primary);
  color: var(--primary-foreground);
  border: none;
  border-radius: var(--radius);
  font-weight: 500;
}

/* Contact info aside */
main:has(> div > form + aside > dl) aside dl {
  display: flex;
  flex-direction: column;
  gap: var(--spacing-6);
}

main:has(> div > form + aside > dl) aside dt {
  font-weight: 600;
  margin-bottom: var(--spacing-2);
}

main:has(> div > form + aside > dl) aside dd {
  color: var(--muted-foreground);
  margin: 0;
}
```

## When to Use

- Contact us pages
- Support inquiry forms
- Business contact information
- Customer service pages
