## Semantic Pattern

A careers page uses `<main>` with header, perks section, and jobs section. The pattern `main:has(> section[aria-label="Perks"] + section[aria-label="Jobs"])` identifies career pages:

```html
<main>
  <header>
    <h1>Join Our Team</h1>
    <p>Help us build the future of web development</p>
  </header>

  <section aria-label="Perks">
    <ul>
      <li>
        <span aria-hidden="true">ğŸŒ</span>
        <h3>Remote First</h3>
        <p>Work from anywhere in the world</p>
      </li>
      <!-- More perks -->
    </ul>
  </section>

  <section aria-label="Jobs">
    <h2>Open Positions</h2>
    <nav aria-label="Departments">
```

## CSS

```css
/* Careers page: main with perks + jobs sections */
main:has(> section[aria-label="Perks"] + section[aria-label="Jobs"]) {
  padding: var(--spacing-6);
}

main:has(> section[aria-label="Perks"]) > header {
  text-align: center;
  max-width: 700px;
  margin: 0 auto var(--spacing-8);
}

main:has(> section[aria-label="Perks"]) > header h1 {
  font-size: var(--text-3xl);
  font-weight: 700;
  margin-bottom: var(--spacing-3);
}

main:has(> section[aria-label="Perks"]) > header p {
  font-size: var(--text-lg);
  color: var(--muted-foreground);
}

/* Perks grid */
section[aria-label="Perks"] {
  margin-bottom: var(--spacing-12);
}

section[aria-label="Perks"] ul {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  gap: var(--spacing-4);
  list-style: none;
  padding: 0;
}

section[aria-label="Perks"] li {
  padding: var(--spacing-4);
  background: var(--muted);
  border-radius: var(--radius-lg);
  text-align: center;
}

section[aria-label="Perks"] li > span {
  font-size: var(--text-2xl);
  display: block;
  margin-bottom: var(--spacing-2);
}

section[aria-label="Perks"] h3 {
  font-weight: 600;
  margin-bottom: var(--spacing-1);
}

section[aria-label="Perks"] p {
  font-size: var(--text-sm);
  color: var(--muted-foreground);
}

/* Job listings */
section[aria-label="Jobs"] h2 {
  font-size: var(--text-xl);
  font-weight: 600;
  margin-bottom: var(--spacing-4);
}

section[aria-label="Jobs"] nav[aria-label="Departments"] {
  display: flex;
  gap: var(--spacing-3);
  margin-bottom: var(--spacing-4);
}

section[aria-label="Jobs"] nav button {
  padding: var(--spacing-2) var(--spacing-4);
  background: var(--muted);
  border: none;
  border-radius: var(--radius-full);
  font-size: var(--text-sm);
}

section[aria-label="Jobs"] nav button[aria-pressed="true"] {
  background: var(--primary);
  color: var(--primary-foreground);
}

  align-items: center;
}

  border-bottom: none;
}

section[aria-label="Jobs"] > ul h3 {
  font-weight: 600;

  border: 1px solid var(--border);
  border-radius: var(--radius);

}
```
