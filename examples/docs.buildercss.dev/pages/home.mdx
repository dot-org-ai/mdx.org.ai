## Semantic Pattern

A landing page uses `<body>` with a global `<header>` containing navigation, followed by `<main>` with multiple `<section>` elements. CSS identifies this as `body:has(> header + main > section:first-child h1)`:

```html
<body>
  <header>
    <a href="/">Acme</a>
    <nav>
      <a href="#features">Features</a>
      <a href="#pricing">Pricing</a>
      <a href="#docs">Docs</a>
    </nav>
    <button>Get Started</button>
  </header>

  <main>
    <section>
      <mark>New: v2.0 Released</mark>
      <h1>Build better products with semantic HTML</h1>
      <p>A design system that uses HTML structure...</p>
      <menu>
        <li><button>Start Building</button></li>
        <li><button>View Docs</button></li>
      </menu>
    </section>

    <section aria-label="Social proof">
      <p>Trusted by teams at</p>
      <ul>
        <li><figure><img src="/logos/vercel.svg" alt="Vercel" /></figure></li>
        <li><figure><img src="/logos/stripe.svg" alt="Stripe" /></figure></li>
      </ul>
    </section>

    <section aria-label="Features">
      <!-- Feature cards -->
    </section>

    <section aria-label="Testimonials">
      <!-- Testimonial cards -->
    </section>

    <section aria-label="Call to action">
      <h2>Ready to get started?</h2>
      <button>Start Free Trial</button>
    </section>
  </main>

  <footer>
    <!-- Footer content -->
  </footer>
</body>
```

## CSS

```css
/* Landing page: body with header + main containing hero section */
body:has(> header + main > section:first-child h1) {
  min-height: 100vh;
}

body:has(> header + main) > header {
  display: flex;
  justify-content: space-between;
  align-items: center;
  padding: var(--spacing-4) var(--spacing-6);
  border-bottom: 1px solid var(--border);
  position: sticky;
  top: 0;
  background: var(--background);
  z-index: 50;
}

body:has(> header + main) > header > a:first-child {
  font-weight: 700;
  font-size: var(--text-xl);
}

body:has(> header + main) > header nav {
  display: flex;
  gap: var(--spacing-6);
  font-size: var(--text-sm);
}

body:has(> header + main) > header nav a {
  color: var(--muted-foreground);
}

body:has(> header + main) > header nav a[aria-current="page"] {
  color: var(--foreground);
}

body:has(> header + main) > header > button {
  padding: var(--spacing-2) var(--spacing-4);
  background: var(--primary);
  color: var(--primary-foreground);
  border: none;
  border-radius: var(--radius);
}

/* Hero section: first section with h1 */
main > section:first-child:has(h1) {
  padding: var(--spacing-16) var(--spacing-6);
  text-align: center;
  max-width: 800px;
  margin: 0 auto;
}

main > section:first-child:has(h1) mark {
  display: inline-block;
  padding: var(--spacing-1) var(--spacing-3);
  background: var(--muted);
  border-radius: var(--radius-full);
  font-size: var(--text-sm);
  margin-bottom: var(--spacing-4);
}

main > section:first-child:has(h1) h1 {
  font-size: var(--text-4xl);
  font-weight: 800;
  line-height: 1.1;
  margin-bottom: var(--spacing-4);
}

main > section:first-child:has(h1) p {
  font-size: var(--text-lg);
  color: var(--muted-foreground);
  margin-bottom: var(--spacing-6);
}

main > section:first-child:has(h1) menu {
  display: flex;
  gap: var(--spacing-3);
  justify-content: center;
  list-style: none;
}

/* Primary: first button */
main > section:first-child:has(h1) menu > :first-child button {
  padding: var(--spacing-3) var(--spacing-6);
  background: var(--primary);
  color: var(--primary-foreground);
  border: none;
  border-radius: var(--radius);
  font-weight: 500;
}

/* Secondary: other buttons */
main > section:first-child:has(h1) menu > :not(:first-child) button {
  padding: var(--spacing-3) var(--spacing-6);
  background: transparent;
  border: 1px solid var(--border);
  border-radius: var(--radius);
  font-weight: 500;
}

/* Social proof section */
section[aria-label="Social proof"] {
  padding: var(--spacing-8) var(--spacing-6);
  background: var(--muted);
}

section[aria-label="Social proof"] p {
  text-align: center;
  font-size: var(--text-sm);
  color: var(--muted-foreground);
  margin-bottom: var(--spacing-4);
}

section[aria-label="Social proof"] ul {
  display: flex;
  justify-content: center;
  gap: var(--spacing-8);
  list-style: none;
  padding: 0;
  opacity: 0.5;
}

section[aria-label="Social proof"] figure {
  font-weight: 600;
  margin: 0;
}
```
