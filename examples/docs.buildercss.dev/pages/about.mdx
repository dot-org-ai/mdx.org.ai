## Semantic Pattern

An about page uses `<main>` with a centered `<header>` followed by story, team, and stats sections. The pattern `main:has(> header + section[aria-label="Story"])` identifies it:

```html
<main>
  <header>
    <h1>About Us</h1>
    <p>We're on a mission to make web development simpler...</p>
  </header>

  <section aria-label="Story">
    <figure><img src="/office.jpg" alt="Our office" /></figure>
    <div>
      <h2>Our Story</h2>
      <p>Founded in 2024...</p>
    </div>
  </section>

  <section aria-label="Team">
    <h2>Our Team</h2>
    <ul>
      <li>
        <article>
          <figure><img src="/avatar.jpg" alt="" /></figure>
          <h3>Alex Chen</h3>
          <p>CEO</p>
        </article>
      </li>
    </ul>
  </section>

  <section aria-label="Stats">
    <dl>
      <div>
        <dd><data value="50000">50K+</data></dd>
        <dt>Active Users</dt>
      </div>
    </dl>
  </section>
</main>
```

## CSS

```css
/* About page: main with header + story section */
main:has(> header + section[aria-label="Story"]) {
  padding: var(--spacing-6);
}

main:has(> header + section[aria-label="Story"]) > header {
  text-align: center;
  max-width: 700px;
  margin: 0 auto var(--spacing-12);
}

main:has(> header + section[aria-label="Story"]) > header h1 {
  font-size: var(--text-3xl);
  font-weight: 700;
  margin-bottom: var(--spacing-4);
}

main:has(> header + section[aria-label="Story"]) > header p {
  font-size: var(--text-lg);
  color: var(--muted-foreground);
}

/* Story section: figure + content */
section[aria-label="Story"] {
  display: grid;
  grid-template-columns: 1fr 1fr;
  gap: var(--spacing-8);
  margin-bottom: var(--spacing-12);
}

section[aria-label="Story"] figure {
  background: var(--muted);
  border-radius: var(--radius-lg);
  aspect-ratio: 16/9;
  margin: 0;
}

section[aria-label="Story"] > div {
  display: flex;
  flex-direction: column;
  justify-content: center;
}

section[aria-label="Story"] h2 {
  font-size: var(--text-xl);
  font-weight: 600;
  margin-bottom: var(--spacing-3);
}

section[aria-label="Story"] p {
  color: var(--muted-foreground);
  margin-bottom: var(--spacing-4);
}

/* Team section: h2 + grid of member cards */
section[aria-label="Team"] {
  margin-bottom: var(--spacing-12);
}

section[aria-label="Team"] h2 {
  font-size: var(--text-xl);
  font-weight: 600;
  margin-bottom: var(--spacing-6);
  text-align: center;
}

section[aria-label="Team"] ul {
  display: grid;
  grid-template-columns: repeat(4, 1fr);
  gap: var(--spacing-6);
  list-style: none;
  padding: 0;
}

section[aria-label="Team"] article {
  text-align: center;
}

section[aria-label="Team"] figure {
  width: 80px;
  height: 80px;
  border-radius: 50%;
  background: var(--muted);
  margin: 0 auto var(--spacing-3);
  display: flex;
  align-items: center;
  justify-content: center;
  font-weight: 600;
}

section[aria-label="Team"] h3 {
  font-weight: 600;
}

section[aria-label="Team"] p {
  font-size: var(--text-sm);
  color: var(--muted-foreground);
}

/* Stats section with data elements */
section[aria-label="Stats"] dl {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  gap: var(--spacing-6);
  text-align: center;
}

section[aria-label="Stats"] dd {
  margin: 0;
}

section[aria-label="Stats"] data {
  font-size: var(--text-3xl);
  font-weight: 700;
  color: var(--primary);
}

section[aria-label="Stats"] dt {
  color: var(--muted-foreground);
}
```

## When to Use

- Company "About Us" page
- Team introduction pages
- Startup story pages
- Organization profile
