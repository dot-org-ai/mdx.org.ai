# 404 Not Found Page

A friendly 404 error page when content is not found.

<Preview>
  <main>
    <data value="404">404</data>
    <h1>Page not found</h1>
    <p>Sorry, we couldn't find the page you're looking for. It might have been moved or deleted.</p>
    <menu>
      <li><a href="#">Go Home</a></li>
      <li><a href="#">Contact Support</a></li>
    </menu>
    <nav aria-label="Suggestions">
      <p>Popular pages:</p>
      <ul>
        <li><a href="#">Documentation</a></li>
        <li><a href="#">Components</a></li>
        <li><a href="#">Blog</a></li>
      </ul>
    </nav>
  </main>
</Preview>

## Semantic Pattern

A 404 page uses `<main>` with a `<data>` element for the error code, heading, message, and navigation options. The pattern `main:has(> data[value="404"])` identifies 404 pages:

```html
<main>
  <data value="404">404</data>
  <h1>Page not found</h1>
  <p>Sorry, we couldn't find the page you're looking for.</p>

  <menu>
    <li><a href="/">Go Home</a></li>
    <li><a href="/contact">Contact Support</a></li>
  </menu>

  <nav aria-label="Suggestions">
    <p>Popular pages:</p>
    <ul>
      <li><a href="/docs">Documentation</a></li>
      <li><a href="/components">Components</a></li>
      <li><a href="/blog">Blog</a></li>
    </ul>
  </nav>
</main>
```

## CSS

```css
/* 404 page: main with data element showing error code */
main:has(> data[value="404"]) {
  padding: var(--spacing-6);
  text-align: center;
  min-height: 100vh;
  display: flex;
  flex-direction: column;
  align-items: center;
  justify-content: center;
}

main:has(> data[value="404"]) > data {
  font-size: 120px;
  font-weight: 800;
  color: var(--muted);
  line-height: 1;
  display: block;
}

main:has(> data[value="404"]) h1 {
  font-size: var(--text-2xl);
  font-weight: 700;
  margin-bottom: var(--spacing-2);
}

main:has(> data[value="404"]) > p {
  color: var(--muted-foreground);
  margin-bottom: var(--spacing-6);
  max-width: 400px;
}

main:has(> data[value="404"]) menu {
  display: flex;
  gap: var(--spacing-3);
  list-style: none;
}

/* Primary: first action */
main:has(> data[value="404"]) menu > :first-child a {
  display: inline-block;
  padding: var(--spacing-3) var(--spacing-6);
  background: var(--primary);
  color: var(--primary-foreground);
  border-radius: var(--radius);
  font-weight: 500;
  text-decoration: none;
}

/* Secondary: other actions */
main:has(> data[value="404"]) menu > :not(:first-child) a {
  display: inline-block;
  padding: var(--spacing-3) var(--spacing-6);
  background: transparent;
  border: 1px solid var(--border);
  border-radius: var(--radius);
  font-weight: 500;
  text-decoration: none;
}

main:has(> data[value="404"]) nav[aria-label="Suggestions"] {
  margin-top: var(--spacing-8);
}

main:has(> data[value="404"]) nav[aria-label="Suggestions"] p {
  font-size: var(--text-sm);
  color: var(--muted-foreground);
  margin-bottom: var(--spacing-3);
}

main:has(> data[value="404"]) nav[aria-label="Suggestions"] ul {
  display: flex;
  gap: var(--spacing-4);
  list-style: none;
  padding: 0;
}

main:has(> data[value="404"]) nav[aria-label="Suggestions"] a {
  font-size: var(--text-sm);
  color: var(--primary);
}
```

## When to Use

- 404 error pages
- Content not found states
- Broken link handling
- Missing resource pages
