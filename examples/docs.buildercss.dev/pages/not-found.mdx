# 404 Not Found Page

A friendly 404 error page when content is not found.

<Preview>
  <article data-page="not-found">
    <data value="404">404</data>
    <h1>Page not found</h1>
    <p>Sorry, we couldn't find the page you're looking for. It might have been moved or deleted.</p>
    <menu>
      <a href="#">Go Home</a>
      <a href="#">Contact Support</a>
    </menu>
    <nav aria-label="Suggestions">
      <p>Popular pages:</p>
      <ul>
        <li><a href="#">Documentation</a></li>
        <li><a href="#">Components</a></li>
        <li><a href="#">Blog</a></li>
      </ul>
    </nav>
  </article>
</Preview>

## When to Use

- 404 error pages
- Content not found states
- Broken link handling
- Missing resource pages

## Structure

```html
<article data-page="not-found">
  <data value="404">404</data>
  <h1>Page not found</h1>
  <p>Sorry, we couldn't find the page you're looking for.</p>

  <menu>
    <a href="/">Go Home</a>
    <a href="/contact">Contact Support</a>
  </menu>

  <nav aria-label="Suggestions">
    <p>Popular pages:</p>
    <ul>
      <li><a href="/docs">Documentation</a></li>
      <li><a href="/components">Components</a></li>
      <li><a href="/blog">Blog</a></li>
    </ul>
  </nav>
</article>
```
