## When to Use

- 500 server error pages
- Application crash states
- Unexpected error handling
- Service unavailable pages

## Structure

```html
<article data-page="error">
  <div data-error-icon>!</div>
  <h1>Something went wrong</h1>
  <p>We're sorry, but something unexpected happened.</p>

  <div data-actions>
    <button data-button="primary" onclick="location.reload()">Try Again</button>
    <a href="/" data-button="outline">Go Home</a>
  </div>

  <details data-error-details>
    <summary>Technical details</summary>
    <div data-error-info>
      <p>Error Code: 500</p>
      <p>Request ID: abc-123-def-456</p>
      <p>Timestamp: 2024-12-01T10:30:00Z</p>
    </div>
  </details>
</article>
```
