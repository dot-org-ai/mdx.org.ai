## Semantic Pattern

A pricing page uses `<main>` with a header and a grid of pricing `<article>` elements containing `<data>` for prices. The pattern `main:has(> div > article data)` identifies pricing comparison:

```html
<main>
  <header>
    <h1>Simple, transparent pricing</h1>
    <p>Choose the plan that's right for you</p>
  </header>
  <div>
    <article>
      <h3>Starter</h3>
      <p>For individuals</p>
      <data value="0">$0</data>
      <span>/month</span>
      <button>Get Started</button>
      <ul>
        <li>Up to 3 projects</li>
        <li>1GB storage</li>
      </ul>
    </article>
    <article aria-current="true">
      <mark>Popular</mark>
      <h3>Pro</h3>
      <p>For growing teams</p>
      <data value="29">$29</data>
      <span>/month</span>
      <button>Start Free Trial</button>
      <ul>...</ul>
    </article>
    <article>
      <h3>Enterprise</h3>
      <p>For large organizations</p>
      <data value="custom">Custom</data>
      <button>Contact Sales</button>
      <ul>...</ul>
    </article>
  </div>
</main>
```

## CSS

```css
/* Pricing page: main with header + div containing pricing articles */
main:has(> header + div > article data) {
  padding: var(--spacing-6);
}

main:has(> header + div > article data) > header {
  text-align: center;
  margin-bottom: var(--spacing-8);
}

main:has(> header + div > article data) > header h1 {
  font-size: var(--text-3xl);
  font-weight: 700;
  margin-bottom: var(--spacing-2);
}

main:has(> header + div > article data) > header p {
  color: var(--muted-foreground);
}

main:has(> header + div > article data) > div {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  gap: var(--spacing-4);
  max-width: 1000px;
  margin: 0 auto;
}

main:has(> header + div > article data) > div > article {
  border: 1px solid var(--border);
  border-radius: var(--radius-lg);
  padding: var(--spacing-6);
}

/* Featured tier: aria-current */
main:has(> header + div > article data) > div > article[aria-current] {
  border: 2px solid var(--primary);
  position: relative;
}

main:has(> header + div > article data) > div > article[aria-current] > mark {
  position: absolute;
  top: -12px;
  left: 50%;
  transform: translateX(-50%);
  background: var(--primary);
  color: var(--primary-foreground);
  padding: 2px 12px;
  border-radius: var(--radius-full);
  font-size: var(--text-xs);
}

main:has(> header + div > article data) article h3 {
  font-weight: 600;
  margin-bottom: var(--spacing-1);
}

main:has(> header + div > article data) article > p {
  font-size: var(--text-sm);
  color: var(--muted-foreground);
  margin-bottom: var(--spacing-4);
}

main:has(> header + div > article data) article data {
  font-size: var(--text-3xl);
  font-weight: 700;
}

main:has(> header + div > article data) article > span {
  color: var(--muted-foreground);
}

main:has(> header + div > article data) article button {
  display: block;
  width: 100%;
  padding: var(--spacing-2);
  margin: var(--spacing-4) 0;
  border-radius: var(--radius);
  cursor: pointer;
}

/* Standard tier buttons */
main:has(> header + div > article data) article:not([aria-current]) button {
  background: transparent;
  border: 1px solid var(--border);
}

/* Featured tier buttons */
main:has(> header + div > article data) article[aria-current] button {
  background: var(--primary);
  color: var(--primary-foreground);
  border: none;
}

main:has(> header + div > article data) article ul {
  list-style: none;
  padding: 0;
  font-size: var(--text-sm);
}

main:has(> header + div > article data) article li {
  padding: var(--spacing-1) 0;
}

main:has(> header + div > article data) article li::before {
```
