## When to Use

- Template galleries
- Product catalogs
- Image galleries
- Portfolio displays
- App stores

## Structure

```html
<article data-view="grid">
  <header>
    <h1>Templates</h1>
    <nav>
      <input type="search" placeholder="Search..." />
      <select>
        <option>All Categories</option>
      </select>
    </nav>
  </header>

  <section>
    <article>
      <figure>
        <img src="/template-preview.jpg" alt="" />
      </figure>
      <div>
        <h3>Landing Page</h3>
        <p>Modern marketing template</p>
        <footer>
          <span>Free</span>
          <button>Use</button>
        </footer>
      </div>
    </article>
  </section>

  <footer>
    <nav><!-- Pagination --></nav>
  </footer>
</article>
```

## CSS

```css
[data-view="grid"] {
  padding: var(--spacing-6);
}

[data-view="grid"] > header {
  display: flex;
  justify-content: space-between;
  align-items: center;
  margin-bottom: var(--spacing-4);
}

[data-view="grid"] > header h1 {
  font-size: var(--text-xl);
  font-weight: 700;
}

[data-view="grid"] > header nav {
  display: flex;
  gap: var(--spacing-2);
}

[data-view="grid"] > header input,
[data-view="grid"] > header select {
  padding: var(--spacing-2);
  border: 1px solid var(--border);
  border-radius: var(--radius);
```
