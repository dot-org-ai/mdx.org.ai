            </div>

## Structure

```html
<article data-view="calendar">
  <header>
    <div data-nav>
      <h1>December 2024</h1>
      <button data-action="prev">←</button>
      <button data-action="next">→</button>
    </div>
    <div data-actions>
      <button data-action="today">Today</button>
      <select data-view-selector>
        <option value="month">Month</option>
        <option value="week">Week</option>
        <option value="day">Day</option>
      </select>
      <button data-button="primary">Add Event</button>
    </div>
  </header>

  <section data-calendar-grid>
    <header data-weekdays>
      <span>Sun</span>
      <span>Mon</span>
      <!-- ... -->
    </header>
    <div data-days>
      <div data-day data-date="2024-12-01">
        <span data-day-number>1</span>
        <div data-event style="--event-color: blue">Team Sync</div>
      </div>
      <!-- ... -->
    </div>
  </section>
</article>
```
