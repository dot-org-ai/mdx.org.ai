```html

  <header>
    <div>
      <h1>Team Members</h1>
      <p>Manage your team's access and permissions</p>
    </div>
    <button data-button="primary">Invite Member</button>
  </header>

  <section data-content>
    <ul data-user-list>
      <li data-user-item>
        <div data-user-info>
          <img src="/avatar.jpg" alt="" data-avatar />
          <div>
            <h3>John Doe</h3>
            <p>john@example.com</p>
          </div>
        </div>
        <div data-user-meta>
          <span data-badge="primary">Admin</span>
          <button data-dropdown-trigger>⋮</button>
        </div>
      </li>
    </ul>
  </section>
</article>
```
