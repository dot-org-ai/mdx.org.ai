# Form View

A form for creating or editing data with validation.

<Preview>
  <article data-view="form">
    <header>
      <h1>Create Project</h1>
    </header>
    <form>
      <div>
        <label>Project Name *</label>
        <input type="text" placeholder="Enter project name" required />
      </div>
      <div>
        <label>Description</label>
        <textarea rows="4" placeholder="Describe your project..."></textarea>
      </div>
      <fieldset>
        <div>
          <label>Category</label>
          <select>
            <option>Select category</option>
            <option>Development</option>
            <option>Design</option>
            <option>Marketing</option>
          </select>
        </div>
        <div>
          <label>Priority</label>
          <select>
            <option>Low</option>
            <option>Medium</option>
            <option>High</option>
          </select>
        </div>
      </fieldset>
      <div>
        <label>
          <input type="checkbox" />
          <span>Make this project public</span>
        </label>
      </div>
    </form>
    <footer>
      <button type="button">Cancel</button>
      <button type="submit">Create Project</button>
    </footer>
  </article>
</Preview>

## When to Use

- Create/edit forms
- Settings forms
- Multi-step wizards
- Data entry screens

## Structure

```html
<article data-view="form">
  <header>
    <h1>Create Project</h1>
  </header>

  <form>
    <div>
      <label for="name">Project Name *</label>
      <input type="text" id="name" required />
      <span role="alert">This field is required</span>
    </div>

    <div>
      <label for="description">Description</label>
      <textarea id="description" rows="4"></textarea>
      <small>Optional description of your project</small>
    </div>

    <fieldset>
      <div>
        <label for="category">Category</label>
        <select id="category">
          <option value="">Select category</option>
          <option value="dev">Development</option>
          <option value="design">Design</option>
        </select>
      </div>
      <div>
        <label for="priority">Priority</label>
        <select id="priority">
          <option value="low">Low</option>
          <option value="medium">Medium</option>
          <option value="high">High</option>
        </select>
      </div>
    </fieldset>

    <div>
      <label>
        <input type="checkbox" name="public" />
        Make this project public
      </label>
    </div>
  </form>

  <footer>
    <button type="button">Cancel</button>
    <button type="submit">Create Project</button>
  </footer>
</article>
```
