## Structure

```html
<article data-view="form">
  <header>
    <h1>Create Project</h1>
  </header>

  <section data-content>
    <form>
      <div data-field>
        <label for="name">Project Name *</label>
        <input type="text" id="name" required />
        <span data-error>This field is required</span>
      </div>

      <div data-field>
        <label for="description">Description</label>
        <textarea id="description" rows="4"></textarea>
        <span data-hint>Optional description of your project</span>
      </div>

      <div data-field-group>
        <div data-field>
          <label for="category">Category</label>
          <select id="category">
            <option value="">Select category</option>
            <option value="dev">Development</option>
            <option value="design">Design</option>
          </select>
        </div>
        <div data-field>
          <label for="priority">Priority</label>
          <select id="priority">
            <option value="low">Low</option>
            <option value="medium">Medium</option>
            <option value="high">High</option>
          </select>
        </div>
      </div>

      <div data-field>
        <label>
          <input type="checkbox" name="public" />
          Make this project public
        </label>
      </div>
    </form>
  </section>

  <footer>
    <button data-button="ghost">Cancel</button>
    <button data-button="primary" type="submit">Create Project</button>
  </footer>
</article>
```

## CSS

```css
[data-view="form"] {
  padding: var(--spacing-6);
  max-width: 600px;
}

[data-view="form"] form {
  display: flex;
  flex-direction: column;
  gap: var(--spacing-4);
}

[data-view="form"] [data-field] {
  display: flex;
  flex-direction: column;
  gap: var(--spacing-1);
}

[data-view="form"] [data-field] label {
  font-weight: 500;
  font-size: var(--text-sm);
}

[data-view="form"] [data-field] input,
[data-view="form"] [data-field] textarea,
[data-view="form"] [data-field] select {
  padding: var(--spacing-2);
  border: 1px solid var(--border);
  border-radius: var(--radius);
}

[data-view="form"] [data-field] input:focus,
[data-view="form"] [data-field] textarea:focus {
  outline: 2px solid var(--ring);
  outline-offset: 2px;
}

[data-view="form"] [data-field] [data-error] {
  color: var(--destructive);
  font-size: var(--text-sm);
  display: none;
}

[data-view="form"] [data-field][data-invalid] [data-error] {
  display: block;
}

[data-view="form"] [data-field-group] {
  display: grid;
  grid-template-columns: 1fr 1fr;
  gap: var(--spacing-4);
}

[data-view="form"] > footer {
  display: flex;
  gap: var(--spacing-2);
  justify-content: flex-end;
  margin-top: var(--spacing-4);
  padding-top: var(--spacing-4);
  border-top: 1px solid var(--border);
}
```
