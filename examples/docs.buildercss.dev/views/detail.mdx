## Semantic Pattern

A detail view is an `<article>` with header and a `<div>` containing `<section>` + `<aside>`:

```html
<article>
  <header>
    <div>
      <mark>Active</mark>
      <time>Created 3 days ago</time>
    </div>
    <h1>Project Alpha</h1>
    <p>Web application redesign project</p>
    <menu>
      <li><button>Edit</button></li>
      <li><button>Delete</button></li>
    </menu>
  </header>

  <div>
    <section>
      <h2>Description</h2>
      <p>Project description...</p>

      <h2>Activity</h2>
      <ul>
        <li>
          <figure><img src="/avatar.jpg" alt="" /></figure>
          <div>
            <p><strong>John</strong> updated status</p>
            <time>2 hours ago</time>
          </div>
        </li>
      </ul>
    </section>

    <aside>
      <h3>Details</h3>
      <dl>
        <div>
          <dt>Owner</dt>
          <dd>John Doe</dd>
        </div>
        <div>
          <dt>Category</dt>
          <dd>Development</dd>
        </div>
      </dl>
    </aside>
  </div>
</article>
```

## CSS

```css
/* Detail view: article with header + div containing section + aside */
article:has(> header + div > section + aside) {
  padding: var(--spacing-6);
}

article:has(> header + div > section + aside) > header {
  display: flex;
  flex-wrap: wrap;
  justify-content: space-between;
  align-items: flex-start;
  margin-bottom: var(--spacing-6);
}

article:has(> header + div > section + aside) > header > div:first-child {
  display: flex;
  align-items: center;
  gap: var(--spacing-2);
  margin-bottom: var(--spacing-2);
  width: 100%;
}

article:has(> header + div > section + aside) > header mark {
  padding: 2px 8px;
  background: green;
  color: white;
  border-radius: var(--radius-full);
  font-size: var(--text-xs);
}

article:has(> header + div > section + aside) > header time {
  font-size: var(--text-sm);
  color: var(--muted-foreground);
}

article:has(> header + div > section + aside) > header h1 {
  font-size: var(--text-2xl);
  font-weight: 700;
  margin-bottom: var(--spacing-1);
  flex: 1;
}

article:has(> header + div > section + aside) > header > p {
  color: var(--muted-foreground);
  width: 100%;
}

article:has(> header + div > section + aside) > header menu {
  display: flex;
  gap: var(--spacing-2);
  list-style: none;
}

}
```
