## Semantic Pattern

A dashboard view is an `<article>` with:
- `<header>` containing title and action menu
- `<section>` containing metric cards with `<data>` elements
- `<div>` with charts/widgets and `<aside>` for activity

```html
<article>
  <header>
    <h1>Dashboard</h1>
    <menu>
      <li><button>Export</button></li>
      <li><button>Settings</button></li>
    </menu>
  </header>

  <!-- Metrics section -->
  <section>
    <article>
      <header>Total Users</header>
      <data value="12345">12,345</data>
      <footer>+12% from last month</footer>
    </article>
    <!-- More metrics -->
  </section>

  <!-- Charts and activity -->
  <div>
    <article>
      <header><h3>Revenue</h3></header>
      <figure><!-- Chart --></figure>
    </article>
    <aside>
      <header><h3>Recent Activity</h3></header>
      <ul><!-- Activity items --></ul>
    </aside>
  </div>
</article>
```
