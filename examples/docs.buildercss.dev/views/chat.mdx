## Structure

```html
<article data-view="chat">
  <header>
    <img src="/avatar.jpg" alt="" data-avatar />
    <div>
      <h2>Support Team</h2>
      <span data-status="online">Online</span>
    </div>
  </header>

  <section data-messages>
    <div data-message data-sender="other">
      <p>Hi! How can I help you today?</p>
      <time>10:30 AM</time>
    </div>

    <div data-message data-sender="self">
      <p>I have a question about my order</p>
      <time>10:31 AM</time>
    </div>
  </section>

  <footer data-input>
    <button data-action="attach">üìé</button>
    <input type="text" placeholder="Type a message..." />
    <button data-button="primary">Send</button>
  </footer>
</article>
```
