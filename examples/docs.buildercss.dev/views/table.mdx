## Structure

```html
<article data-view="table">
  <header>
    <h1>Users</h1>
    <div data-actions>
      <input type="search" placeholder="Search..." />
      <button data-button="primary">Add User</button>
    </div>
  </header>

  <section data-content>
    <table data-table>
      <thead>
        <tr>
          <th data-sortable>Name</th>
          <th data-sortable>Email</th>
          <th>Role</th>
          <th>Status</th>
          <th>Actions</th>
        </tr>
      </thead>
      <tbody>
        <tr>
          <td>
            <div data-user>
              <img src="/avatar.jpg" alt="" />
              <span>John Doe</span>
            </div>
          </td>
          <td>john@example.com</td>
          <td>Admin</td>
          <td><span data-badge="success">Active</span></td>
          <td>
            <button data-dropdown-trigger>⋮</button>
          </td>
        </tr>
      </tbody>
    </table>
  </section>

  <footer>
    <span>Showing 1-10 of 100</span>
    <nav data-pagination>...</nav>
  </footer>
</article>
```

## CSS

```css
[data-view="table"] {
  padding: var(--spacing-6);
}

[data-view="table"] [data-table] {
  width: 100%;
  border-collapse: collapse;
}

[data-view="table"] [data-table] thead {
  background: var(--muted);
}

[data-view="table"] [data-table] th {
  padding: var(--spacing-3);
  text-align: left;
  font-weight: 500;
  font-size: var(--text-sm);
}

[data-view="table"] [data-table] th[data-sortable] {
  cursor: pointer;
}

[data-view="table"] [data-table] th[data-sortable]:hover {
  background: var(--accent);
}

[data-view="table"] [data-table] td {
  padding: var(--spacing-3);
  border-top: 1px solid var(--border);
}

[data-view="table"] [data-table] tr:hover {
  background: var(--accent);
}
```
