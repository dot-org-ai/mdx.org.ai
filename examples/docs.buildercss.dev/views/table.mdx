## Semantic Pattern

A table view is an `<article>` containing a `<table>` element:

```html
<article>
  <header>
    <h1>Users</h1>
    <nav>
      <input type="search" placeholder="Search..." />
      <button>Add User</button>
    </nav>
  </header>

  <table>
    <thead>
      <tr>
        <th>Name</th>
        <th>Email</th>
        <th>Role</th>
        <th>Status</th>
        <th>Actions</th>
      </tr>
    </thead>
    <tbody>
      <tr>
        <td>
          <figure><img src="/avatar.jpg" alt="" /></figure>
          <span>John Doe</span>
        </td>
        <td>john@example.com</td>
        <td>Admin</td>
        <td><mark>Active</mark></td>
        <td><button aria-label="Actions">⋮</button></td>
      </tr>
    </tbody>
  </table>

  <footer>
    <span>Showing 1-10 of 100</span>
    <nav><!-- Pagination --></nav>
  </footer>
</article>
```

  padding: var(--spacing-6);

  margin-bottom: var(--spacing-4);

  gap: var(--spacing-2);
