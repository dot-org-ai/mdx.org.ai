## Structure

```html
<article data-view="editor">
  <header>
    <div data-file-info>
      <span data-filename>index.ts</span>
      <span data-status>Modified</span>
    </div>
    <div data-actions>
      <button>Format</button>
      <button data-button="primary">Save</button>
    </div>
  </header>

  <div data-editor-container>
    <aside data-line-numbers>
      <span>1</span>
      <span>2</span>
      <!-- ... -->
    </aside>
    <div data-editor>
      <!-- Monaco or CodeMirror instance -->
    </div>
  </div>

  <footer data-status-bar>
    <span data-language>TypeScript</span>
    <span data-position>Ln 3, Col 12</span>
  </footer>
</article>
```

## Integration with Monaco

```tsx
import * as monaco from 'monaco-editor'

function Editor({ value, language, onChange }) {
  const containerRef = useRef(null)
  const editorRef = useRef(null)

  useEffect(() => {
    editorRef.current = monaco.editor.create(containerRef.current, {
      value,
      language,
      theme: 'vs-dark',
      minimap: { enabled: false },
      automaticLayout: true,
    })

    editorRef.current.onDidChangeModelContent(() => {
      onChange?.(editorRef.current.getValue())
    })

    return () => editorRef.current?.dispose()
  }, [])

  return <div ref={containerRef} data-editor />
}
```
