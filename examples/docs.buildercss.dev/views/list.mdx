## Semantic Pattern

A list view is an `<article>` containing a `<ul>` with list items and a pagination footer:

```html
<article>
  <header>
    <h1>Projects</h1>
    <nav>
      <input type="search" placeholder="Search..." />
      <select>
        <option>All Status</option>
        <option>Active</option>
        <option>Archived</option>
      </select>
      <button>New Project</button>
    </nav>
  </header>

  <ul>
    <li>
      <div>
        <h3>Project Alpha</h3>
        <p>Description of the project</p>
        <footer>
          <mark>Active</mark>
          <time>Updated 2 hours ago</time>
        </footer>
      </div>
      <menu>
        <li><button>Edit</button></li>
        <li><button>Delete</button></li>
      </menu>
    </li>
  </ul>

  <footer>
    <span>Showing 1-10 of 50</span>
    <nav>
      <button disabled>Previous</button>
      <button>Next</button>
    </nav>
  </footer>
</article>
```
