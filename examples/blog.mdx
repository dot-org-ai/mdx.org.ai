# Getting Started with MDX

MDX is a powerful format that lets you write JSX in your Markdown documents.
This blog post demonstrates how to use MDX with embedded documentation.

## What is MDX?

MDX allows you to:

- Write Markdown content
- Embed React components
- Include executable code examples

## Basic Example

Here's a simple greeting component:

```jsx
function Greeting({ name }) {
  return <h1>Hello, {name}!</h1>
}

// Usage
<Greeting name="World" />
```

## String Utilities

When building applications, you often need string utilities.

### Capitalize Function

A simple function to capitalize the first letter:

```ts
export function capitalize(str: string): string {
  if (!str) return ''
  return str.charAt(0).toUpperCase() + str.slice(1)
}
```

### Slug Generation

Converting titles to URL-friendly slugs:

```ts
export function slugify(text: string): string {
  return text
    .toLowerCase()
    .trim()
    .replace(/[^\w\s-]/g, '')
    .replace(/[\s_-]+/g, '-')
    .replace(/^-+|-+$/g, '')
}
```

MDX makes it easy to create rich, interactive documentation.
